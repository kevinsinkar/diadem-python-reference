# ADMAdmin

!!! abstract "Command &middot; `ComOnl.chm`"
    Command: ADMAdmin

Opens the dialog box for user management. If the user is not logged on in the user management, DIAdem opens the dialog box for logging on.

## Signature

```python
dd.ADMAdmin()
```
