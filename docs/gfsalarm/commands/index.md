# Commands

8 commands from `Gfsalarm.chm`.

| Name | Summary |
| --- | --- |
| [`PkAlFmtAdd`](pkalfmtadd.md) | Appends the character string that is transferred as a parameter, to the current format string. |
| [`PkAlFMTClear`](pkalfmtclear.md) | Deletes the current format string. Use the PkAlFMTAdd command to create new format strings. |
| [`PkAlFootAdd`](pkalfootadd.md) | Adds a string to the format string for the end text of the logfile. DIAdem adds this string each time a measurement starts. |
| [`PkAlFootClear`](pkalfootclear.md) | Deletes the format string for the end text of the logfile. |
| [`PkAlHeadAdd`](pkalheadadd.md) | Adds a string to the format string for the start text that DIAdem writes to the logfile when a measurement starts. |
| [`PkAlHeadClear`](pkalheadclear.md) | Deletes the format string for the start text that DIAdem writes to the logfile when a measurement starts. |
| [`PkAlResAdd`](pkalresadd.md) | Adds a string to the format string that DIAdem writes to the logfile after the logfile is reset. |
| [`PkAlResClear`](pkalresclear.md) | Deletes the format string that DIAdem writes to the logfile after the logfile is reset. |
