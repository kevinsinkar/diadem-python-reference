# Collections

12 collections from `DataPlugin.chm`.

| Name | Summary |
| --- | --- |
| [`AbstractDirectAccessChannels`](abstractdirectaccesschannels.md) | The AbstractDirectAccessChannels collection corresponds to a DirectAccessChannels or a BinaryDirectAccessChannels collection. |
| [`BinaryDirectAccessChannels`](binarydirectaccesschannels.md) | Collection of all DirectAccess channels. You can use the BinaryDirectAccessChannels collection to delete or to add DirectAccess channels. |
| [`ChannelGroups`](channelgroups.md) | Collection of all channel groups . Use the ChannelGroups collection to delete channel groups or to add new channel groups. |
| [`Channels`](channels.md) | Collection of all channels . Use the Channels collection to delete channels or to add new channels. |
| [`ChannelsToProcess`](channelstoprocess.md) | Collection of all the channels that a Processed channel comprises. Use the ChannelsToProcess collection to delete channels or to add new channels. |
| [`DirectAccessChannels`](directaccesschannels.md) | Collection of all DirectAccess channels. You can use the DirectAccessChannels collection to delete or to add DirectAccess channels. |
| [`DirectCellChannels`](directcellchannels.md) | Collection of all DirectAccess channels. You can use the DirectCellChannels collection to delete or to add DirectAccess channels. |
| [`DocumentNamespaces`](documentnamespaces.md) | Collection of name spaces in a document. Use namespaces to group XML elements in a document. |
| [`DocumentProperties`](documentproperties.md) | Collection of a document's meta properties not allocated to any namespace. A document might be an Office file in XML format or a PDF, JPEG, PNG, or TIFF file. |
| [`Elements`](elements.md) | Collection of all Element objects. These objects contain information about the currently edited data store. The elements from this collection might be ChannelGroup or Channel types. |
| [`Properties`](properties.md) | A collection of all properties for the Root object, for the ChannelGroup object, or for the Channel object. The properties include the base properties of the data model and user-defined custom properties. You can use the Properties collection to delete or to add properties. |
| [`Sheets`](sheets.md) | Collection of all Worksheets . Use the sheets collection to access individual worksheets of the open work folder. |
