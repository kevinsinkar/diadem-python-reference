# Properties

7 properties from `ContextMenu.chm`.

| Name | Summary |
| --- | --- |
| [`IMenuPoint.Check`](imenupoint-check.md) | Specifies whether a context menu item is active. DIAdem identifies the enabled items with a tick in front of the item. |
| [`IMenuPoint.Enable`](imenupoint-enable.md) | Specifies whether DIAdem disables a context menu item. |
| [`IMenuPoint.ID`](imenupoint-id.md) | Specifies the ID of a context menu item. Use the ID to determine which action DIAdem executes in the OnContextMenuPointSelected event. |
| [`IMenuPoint.MenuPoints`](imenupoint-menupoints.md) | Returns the items of a submenu. |
| [`IMenuPoint.Tag`](imenupoint-tag.md) | Contains additional information about the properties of a context menu item. Use the Tag property in the OnContextMenuPointSelected event. |
| [`IMenuPoint.Text`](imenupoint-text.md) | Specifies the text of the context menu item. |
| [`IMenuPoints.Count`](imenupoints-count.md) | Specifies the number of entries in the context menu. |
