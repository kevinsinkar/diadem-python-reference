# IDiademPropertyList.Add

!!! abstract "Method &middot; `Inavidata.chm`"
    Method: Add for PropertyList <Data>

Creates a new element in the script interface for internal data and adds this element to the PropertyList collection. The Add method returns an Element object. This method is not available if the PropertyList collection is read-only.

## Signature

```python
return_value = obj.Add(DataProperty)
```
