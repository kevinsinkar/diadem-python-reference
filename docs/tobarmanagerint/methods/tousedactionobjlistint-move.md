# ToUsedActionObjListInt.Move

!!! abstract "Method &middot; `ToBarManagerInt.chm`"
    Method: Move for UsedActionObjs

Changes the position of an element in a bar. If you change the position, you also change the index of the element.

## Signature

```python
obj.Move(FromPos, ToPos)
```

## Python example

```python
dd.BarManager.Bars("MyBar").UsedActionObjs.Move(1,3)
```
