# Collections

5 collections from `ToBarManagerInt.chm`.

| Name | Summary |
| --- | --- |
| [`ActionObjs`](actionobjs.md) | Collection of all bar elements in DIAdem. To access an individual bar element in a script, use the Item property, or enter the index or the name in parentheses. |
| [`Bars`](bars.md) | Collection of all bars that you can use in DIAdem. To access an individual bar in a script use the Item method or enter the index or the name in parentheses. |
| [`Panels`](panels.md) | A collection of all panels available in DIAdem. To access an individual panel in a script, use the Item method, or enter the index or the name in parentheses. |
| [`UsedActionObjs`](usedactionobjs.md) | Collection of all elements used in a bar. The elements that a bar contains are references to elements of the ActionObj collection. When you access a UsedActionObjs collection you receive the element of the ActionObjs collection. To access an individual bar element in a script, use the Item method, or enter the index or the name in parentheses. |
| [`UsedBars`](usedbars.md) | Collection of all bars used in a DIAdem panel or in the DIAdem main window. The bars that a panel or the DIAdem main window contain are references to bars of the Bars collection. To access an individual bar in a script use the Item method or enter the index or the name in parentheses. |
