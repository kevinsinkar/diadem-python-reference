# Properties

12 properties from `LVHelper.chm`.

| Name | Summary |
| --- | --- |
| [`ILVRuntime.Version`](ilvruntime-version.md) | Returns the LabVIEW version that the LVRuntime uses. |
| [`ITerminal.Dimensions`](iterminal-dimensions.md) | Returns the dimension of a terminal. A terminal is an input terminal or an output terminal of a VI. |
| [`ITerminal.IsOut`](iterminal-isout.md) | Specifies whether a terminal is an output. |
| [`ITerminal.Name`](iterminal-name.md) | Returns the name of an incoming and outgoing VI terminal. |
| [`ITerminal.Type`](iterminal-type.md) | Returns the type of an incoming and outgoing VI terminal. |
| [`ITerminals.Count`](iterminals-count.md) | Returns the number of VI terminals. A terminal is an input terminal or an output terminal of a VI. |
| [`IVIRuntime.Description`](iviruntime-description.md) | Returns the description of a VI. |
| [`IVIRuntime.HelpPath`](iviruntime-helppath.md) | Returns the path for the HTML file ( *.htm or *.html ) or the help file ( *.chm or *.hlp ) for which the VI contains a link. If the path leads to a compiled help file, the HelpTag property enables the selection of a topic in the help file. This property is similar to the Help path textbox from the Documentation page in the LabVIEW VI properties dialog field. |
| [`IVIRuntime.HelpTag`](iviruntime-helptag.md) | Returns an index keyword or an HTML filename for a topic in the compiled help file to which the VI contains a link. For *.chm files this property can be an HTML filename or an index keyword. For *.hlp files this property can be an index keyword. This property is similar to the Help tag textbox from the Documentation page in the LabVIEW VI properties dialog field. |
| [`IVIRuntime.Name`](iviruntime-name.md) | Returns the complete name and path of the VI. |
| [`IVIRuntime.Terminals`](iviruntime-terminals.md) | Returns a list of output terminals and input terminals of a VI. |
| [`IVIRuntime.Version`](iviruntime-version.md) | Returns the LabVIEW version with which the VI was generated. |
