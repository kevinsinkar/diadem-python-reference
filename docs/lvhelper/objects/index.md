# Objects

3 objects from `LVHelper.chm`.

| Name | Summary |
| --- | --- |
| [`ILVRuntime`](ilvruntime.md) | The LVRuntime object enables access to LabVIEW. Use the LVRuntime object to load and to execute VIs. The LVRuntime object is available as a global object in scripts and in dialog boxes. Do not create an object or a variable with the name LVRuntime , because that overwrites the Navigator object. |
| [`ITerminal`](iterminal.md) | Returns a Terminal object. A terminal is an input terminal or an output terminal of a VI. |
| [`IVIRuntime`](iviruntime.md) | The VIRuntime object enables you to access LabVIEW VIs from DIAdem. Use the VIRuntime object to access VI properties such as the name or the list of terminals. You also set values of the VIs or execute VIs. |
