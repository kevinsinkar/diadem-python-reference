# Collections

3 collections from `FileTagging.chm`.

| Name | Summary |
| --- | --- |
| [`DocumentNamespaces`](documentnamespaces.md) | Collection of name spaces in a document. Use namespaces to identify XML elements clearly in a document. A document may be an Office file in XML format or a PDF, JPEG, PNG, or TIFF file, which uses XMP. Use the OpenDocument command to open a document. |
| [`DocumentProperties`](documentproperties.md) | Collection of changeable meta properties that are not assigned to a namespace, from the top level of a document. A document may be an Office file in XML format or a PDF, JPEG, PNG, or TIFF file, which uses XMP. Use the OpenDocument command to open a document. The five properties name , description , title , author , and datetime are the base properties of a document. They are permanent and cannot be deleted. |
| [`DocumentReadOnlyProperties`](documentreadonlyproperties.md) | Collection of all meta properties in a specific namespace of a document. The meta properties in a specific namespace are read-only. Use namespaces to identify XML elements clearly in a document. A document may be an Office file in XML format or a PDF, JPEG, PNG, or TIFF file, which uses XMP. Use the OpenDocument command to open a document. |
