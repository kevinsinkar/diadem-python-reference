# Objects

3 objects from `FileTagging.chm`.

| Name | Summary |
| --- | --- |
| [`IDiademDocument`](idiademdocument.md) | The Document <Data> object provides the properties and methods for accessing the meta properties of a document. A document may be an Office file in XML format or a PDF, JPEG, PNG, or TIFF file, which uses XMP. Use the OpenDocument command to open a document. |
| [`IDocumentNamespace`](idocumentnamespace.md) | The DocumentNamespace <Data> object provides the namespace in a document. Use namespaces to identify XML elements clearly in a document. A document may be an Office file in XML format or a PDF, JPEG, PNG, or TIFF file, which uses XMP. Use the OpenDocument command to open a document. |
| [`IDocumentProperty`](idocumentproperty.md) | The DocumentProperty <Data> object provides the meta properties of the document. A document may be an Office file in XML format or a PDF, JPEG, PNG, or TIFF file, which uses XMP. Use the OpenDocument command to open a document. |
