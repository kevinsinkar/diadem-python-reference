# ITDMCell.Column

!!! abstract "Property &middot; `ScriptNavi.chm`"
    Property: Column for Cell

Specifies in the search results list of a DataFinder or a data store a column with properties.

## Signature

```python
return_value = obj.Column
```

## Python example

```python
oMyResultsList = dd.Navigator.Display.CurrDataFinder.ResultsList
oMyCell = oMyResultsList.FocusedCell
if (not oMyCell is None) and (not oMyCell.Element is None) :
    oMyElementVal = oMyCell.Element.Properties(oMyCell.Column.Property).value
    oMyText = oMyCell.Column.Property + "=" + oMyElementVal
    dd.MsgBoxDisp(oMyText)
```
