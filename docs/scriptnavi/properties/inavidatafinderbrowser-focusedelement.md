# INaviDataFinderBrowser.FocusedElement

!!! abstract "Property &middot; `ScriptNavi.chm`"
    Property: FocusedElement for Browser <DataFinder>

Specifies in the file browser of a DataFinder the element that has the focus.

## Signature

```python
return_value = obj.FocusedElement
```

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img height="25" src="../image/note.gif" width="26"/></td><td><strong>Note</strong>  The focus is always enabled and only for one element at a time.</td></tr></table>
</div>

## Python example

```python
dd.Navigator.LoadData(Navigator.Display.CurrDataFinder.Browser.FocusedElement)
```
