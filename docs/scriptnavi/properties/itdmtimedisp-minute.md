# ITDMTimeDisp.Minute

!!! abstract "Property &middot; `ScriptNavi.chm`"
    Property: Minute for UsiTimeDisp

Specifies the minutes in a UsiTimeDisp type object.

## Signature

```python
obj.Minute
```

