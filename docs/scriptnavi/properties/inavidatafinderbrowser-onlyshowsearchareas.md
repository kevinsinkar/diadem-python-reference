# INaviDataFinderBrowser.OnlyShowSearchAreas

!!! abstract "Property &middot; `ScriptNavi.chm`"
    Property: OnlyShowSearchAreas for Browser <DataFinder>

Specifies in the file browser of a DataFinder whether only search areas or additionally the local file system is displayed.

## Signature

```python
obj.OnlyShowSearchAreas
```

## Python example

```python
dd.MsgBoxDisp(dd.Navigator.Display.CurrDataFinder.Browser.OnlyShowSearchAreas)
```

```python
dd.Navigator.Display.CurrDataFinder.Browser.OnlyShowSearchAreas = True
```
