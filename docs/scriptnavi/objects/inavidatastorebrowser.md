# INaviDataStoreBrowser

!!! abstract "Object &middot; `ScriptNavi.chm`"
    Object: Browser <DataStore>

The Browser object provides the data browser of a data store. You can use the data browser to navigate tests, SubTests, measurements, and measurement quantities.

## Python example

```python
dd.Navigator.Display.CurrDataStore.Browser.Activate()
```
