# ITDMDataFinderExcludeFolders.ItemByPath

!!! abstract "Method &middot; `ScriptNavi.chm`"
    Method: ItemByPath for ExcludeFolders

Returns the ExcludeFolder object of a specific path, in a DataFinder configuration.

## Signature

```python
return_value = obj.ItemByPath(Path)
```

## Python example

```python
oMySettings = dd.Navigator.ConnectDataFinder("My DataFinder").GetSettings()
oMySearchArea = oMySettings.SearchAreas("MyFirstSearchArea")
oMyExcludedFolder = oMySearchArea.ItemByPath("C:\\MyFirstExcludedFolder")
dd.MsgBoxDisp(oMyExcludedFolder.Name)
```
