# IPropertyImportProperties.Remove

!!! abstract "Method &middot; `ScriptNavi.chm`"
    Method: Remove for ImportProperties

Removes an element from the Loading Configuration of Data Stores . An element consists of a property and the property type. You only can remove those elements from the loading configuration which DIAdem does not import by default when loading data.

## Signature

```python
obj.Remove(Index)
```

## Python example

```python
oMyPropertyImportSet = dd.Navigator.Display.CurrDataStore.PropertyImportSet
oMyImportProperties = oMyPropertyImportSet.Properties
oMyImportProperties.Remove(5)
```
