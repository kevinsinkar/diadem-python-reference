# INaviDataFinderQueryForm.Search

!!! abstract "Method &middot; `ScriptNavi.chm`"
    Method: Search for QueryForm <DataFinder>

Executes an interface search with the current search conditions in a DataFinder. Specify the search conditions before the search starts with SetCurrQuery , or Conditions . Add or enter them manually on the DataFinder interface.

## Signature

```python
obj.Search()
```

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img src="../image/note.gif"/></td><td><strong>Note</strong>  DIAdem saves the results of a search on the DIAdem interface in the <a href="../../objects/inavidatafinderresultdisplay/">ResultsList</a> object.</td></tr></table>
</div>

## Python example

```python
oMyDataFinder = dd.Navigator.Display.CurrDataFinder
oMyDataFinder.QueryForm.Search()
```
