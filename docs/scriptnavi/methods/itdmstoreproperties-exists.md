# ITDMStoreProperties.Exists

!!! abstract "Method &middot; `ScriptNavi.chm`"
    Method: Exists for Properties <DataStore>

Checks in a data store whether the property with the specified name already exists.

## Signature

```python
bExists = Object.Exists(sName)
```

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img src="../image/note.gif"/></td><td><strong>Note  </strong>Assign the value TRUE to the <a href="../../properties/itdmstoreproperties-listinstanceproperties/">ListInstanceProperties</a> property in order read or to edit instance properties. Instance properties are user-defined properties.</td></tr></table>
</div>

## Python example

```python
oMyDataStore = dd.Navigator.ConnectDataStore("ASAM Browse Settings Example")
oMyProperties = oMyDataStore.RootElements(1).Properties
dd.MsgBoxDisp(oMyProperties.Exists("test"))
```
