# ITDMStoreElements.Item

!!! abstract "Method &middot; `ScriptNavi.chm`"
    Method: Item for Elements <DataStore>

Returns the Elements object associated with a specific name or with a specific index, in a data store.

## Signature

```python
return_value = obj.Item(NameOrIndex)
```

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img src="../image/note.gif"/></td><td><strong>Note  </strong>You can always omit the <span class="Monospace">Item</span> method because this method is always the standard element of the collection.</td></tr></table>
</div>

## Python example

```python
oMyDataStore = dd.Navigator.ConnectDataStore("ASAM Browse Settings Example")
oMyRootElement = oMyDataStore.RootElements(1)
oMyStoreElements = oMyRootElement.Children
dd.MsgBoxDisp(oMyStoreElements.Item(1).Name)
```
