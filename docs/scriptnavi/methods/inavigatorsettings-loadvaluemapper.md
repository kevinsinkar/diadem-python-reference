# INavigatorSettings.LoadValueMapper

!!! abstract "Method &middot; `ScriptNavi.chm`"
    Method: LoadValueMapper for Settings <Navigator>

Loads a ValueMapping file. In a mapping you use rules to replace property values. Mapping files have the filename extension *.TPVM (Technical Property Value Mapping).

## Signature

```python
return_value = obj.LoadValueMapper(FileName)
```

## Python example

```python
dd.Navigator.Settings.LoadValueMapper("D:\\MyValueMapper.tpmv")
```
