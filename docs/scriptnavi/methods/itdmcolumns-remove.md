# ITDMColumns.Remove

!!! abstract "Method &middot; `ScriptNavi.chm`"
    Method: Remove for Columns

Removes, in a DataFinder or a data store, a column from the search results.

## Signature

```python
obj.Remove(Index)
```

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img height="25" src="../image/note.gif" width="26"/></td><td><strong>Note  </strong>If you run a search, DIAdem automatically generates new columns. You only can modify the columns that you have added manually.</td></tr></table>
</div>

## Python example

```python
oMyResultsList = dd.Navigator.Display.CurrDataFinder.ResultsList
Index = 1
for Column in oMyResultsList.Columns:
    if Column.Type == dd.eSearchChannel :
        oMyResultsList.Columns.Remove(Index)
    else:
        Index = Index + 1
```
