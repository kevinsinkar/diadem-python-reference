# ITDMStoreElement.Save

!!! abstract "Method &middot; `ScriptNavi.chm`"
    Method: Save for Element <DataStore>

Saves a data element in the corresponding data store.

## Signature

```python
obj.Save()
```

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img src="../image/note.gif"/></td><td><strong>Note</strong>  If you use the <span class="Monospace">Save</span> method for file-based data stores, you must save the data store separately.</td></tr></table>
</div>

## Python example

```python
oMyDataStore = dd.Navigator.ConnectDataStore("ASAM Browse Settings Example")
oMyElement = oMyDataStore.RootElements(1)
oMyElement.Save()
```
