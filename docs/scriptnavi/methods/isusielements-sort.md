# ISUSIElements.Sort

!!! abstract "Method &middot; `ScriptNavi.chm`"
    Method: Sort for ElementList <DataFinder>

Sorts in a DataFinder the collection of the search results alphabetically in ascending or descending order.

## Signature

```python
obj.Sort([Property], [Sorting])
```

## Python example

```python
oMyResultsList = dd.Navigator.Display.CurrDataFinder.ResultsList
oMyResultsList.Elements.Sort(oMyResultsList.Columns(1).GetPath(oMyResultsList.Elements(1).Type),False)
```
