# ImportProperties

!!! abstract "Collection &middot; `ScriptNavi.chm`"
    Collection: ImportProperties

Elements collection of a Loading Configuration of Data Stores . An element consists of a property and the property type.

## Python example

```python
oMyPropertyImportSet = dd.Navigator.Display.CurrDataStore.PropertyImportSet
oMyImportProperties = oMyPropertyImportSet.Properties
oMyImportProperties.Add("MeaQuantity", "Minimum")
```
