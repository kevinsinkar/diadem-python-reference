# PkChS

!!! abstract "Variable &middot; `PkVarOnl.chm`"
    Variable: PkChS

Specifies in packet processing whether DIAdem writes data into one data channel or into several data channels. DIAdem creates the new data channels in the default group of the Data Portal.
