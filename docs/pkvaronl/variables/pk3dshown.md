# Pk3DShowN

!!! abstract "Variable &middot; `PkVarOnl.chm`"
    Variable: Pk3DShowN

Specifies in the packet processing oscilloscope the number of curves to plot in a waterfall diagram or the number values to be displayed in color in the color/campbell diagram.
