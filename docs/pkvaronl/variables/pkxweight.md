# PkXWeight

!!! abstract "Variable &middot; `PkVarOnl.chm`"
    Variable: PkXWeight

Specifies in packet processing that DIAdem evaluates time related and frequency related data such as the results of an FFT or a third analysis with a digital filter order in accordance with DIN IEC 651.
