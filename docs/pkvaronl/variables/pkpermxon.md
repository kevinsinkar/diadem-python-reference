# PkPermXOn

!!! abstract "Variable &middot; `PkVarOnl.chm`"
    Variable: PkPermXOn

Specifies in packet processing a fixed scaling for the x-axis of the oscilloscope. The permanent scaling of the x-axis is possible only for y/x-diagrams.
