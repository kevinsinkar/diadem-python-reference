# Commands

10 commands from `PkVarOnl.chm`.

| Name | Summary |
| --- | --- |
| [`PkBmpClip`](pkbmpclip.md) | Copies the current signal display in the oscilloscope display window as a bitmap to the clipboard. |
| [`PkBMPPrint`](pkbmpprint.md) | Prints the current signal display in the oscilloscope display window as a bitmap. |
| [`PkChNameSet`](pkchnameset.md) | Renames a data channel displayed in the oscilloscope. |
| [`PkCursSwitch`](pkcursswitch.md) | Shows and hides two cursor lines for the measurement of curves in the oscilloscope display window and the associated measurement data window . |
| [`PkDiagrClip`](pkdiagrclip.md) | Copies the current signal display in the oscilloscope display window as a Windows metafile to the clipboard. |
| [`PkDiagrPrint`](pkdiagrprint.md) | Prints the current signal display in the oscilloscope display window as a vector graphic. |
| [`PkPhMaxReset`](pkphmaxreset.md) | Displays the penultimate greatest measurement value of the drag pointer in the voltmeter. |
| [`PkPhMinReset`](pkphminreset.md) | Displays the penultimate smallest measurement value of the drag pointer in the voltmeter. |
| [`PkRecReset`](pkrecreset.md) | Deletes the current display in the recorder mode of the oscilloscope display window. |
| [`PkSendCurs`](pksendcurs.md) | Outputs the y-values to the left cursor line via data output M. when the measurement window is open. The command only takes effect after measurement start. |
