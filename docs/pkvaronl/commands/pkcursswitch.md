# PkCursSwitch

!!! abstract "Command &middot; `PkVarOnl.chm`"
    Command: PkCursSwitch

Shows and hides two cursor lines for the measurement of curves in the oscilloscope display window and the associated measurement data window .

## Python example

```python
PkCursSwitch()
```
