# IFormulaAdaptor.AdaptorType

!!! abstract "Property &middot; `ICalculationSet.chm`"
    Property: AdaptorType for InputOrOutput

Specifies the data type of an input or an output of a calculation.

## Signature

```python
obj.AdaptorType
```

## Enumeration values

| Name | Value | Description |
| --- | ---: | --- |
| `eAdaptorTypeValue` | 100 | The input or the output refers to a value. |
| `eAdaptorTypeChannel` | 101 | The input or the output refers to a channel. |
| `eAdaptorTypeChannelList` | 102 | The input or the output refers to a channel list. |
