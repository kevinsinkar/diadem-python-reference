# IToEventsInt.OnLegendClicked

!!! abstract "Property &middot; `Scriptview.chm`"
    Property: OnLegendClicked for Events

Specifies the name of the user command that DIAdem executes when you click a legend in DIAdem VIEW. The event is triggered when you left-click the legend of an area.

## Signature

```python
obj.OnLegendClicked
```
