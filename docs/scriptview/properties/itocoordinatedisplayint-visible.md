# IToCoordinateDisplayInt.Visible

!!! abstract "Property &middot; `Scriptview.chm`"
    Property: Visible for CoordinateWnd

Specifies whether the coordinate window is visible in DIAdem VIEW.

## Signature

```python
obj.Visible
```

## Python example

```python
dd.View.CoordinateWnd.SetPos(300,300)
dd.View.CoordinateWnd.Visible = True
```
