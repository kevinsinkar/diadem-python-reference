# IToPictureInt.FileName

!!! abstract "Property &middot; `Scriptview.chm`"
    Property: FileName for Picture

Specifies the name of the graphic file in DIAdem VIEW.

## Signature

```python
obj.FileName
```

## Python example

```python
dd.View.ActiveSheet.ActiveArea.DisplayObjType = "Picture"
oMyObj = dd.View.ActiveSheet.ActiveArea.DisplayObj
oMyObj.FileName = "C:\\Pictures\\Example.wmf"
```
