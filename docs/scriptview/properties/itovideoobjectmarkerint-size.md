# IToVideoObjectMarkerInt.Size

!!! abstract "Property &middot; `Scriptview.chm`"
    Property: Size for MarkerObject

In DIAdem VIEW, specifies the size of a marker type OverlayObject object as a percentage of the video area. DIAdem only takes this setting into account if you have not specified a width or height channel .

## Signature

```python
obj.Size
```
