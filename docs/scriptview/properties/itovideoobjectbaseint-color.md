# IToVideoObjectBaseInt.Color

!!! abstract "Property &middot; `Scriptview.chm`"
    Property: Color for OverlayObject

Specifies the color of an OverlayObject object in a video in DIAdem VIEW. If you specify colors using the ColorChannelName property, DIAdem ignores this color.

## Signature

```python
obj.Color
```
