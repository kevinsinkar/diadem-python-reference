# IToVideoInt.Sound

!!! abstract "Property &middot; `Scriptview.chm`"
    Property: Sound for Video

Specifies whether the sound is enabled when playing a video in DIAdem VIEW.

## Signature

```python
obj.Sound
```

## Python example

```python
dd.View.ActiveSheet.ActiveArea.DisplayObjType = "Video"
oMyObj = dd.View.ActiveSheet.ActiveArea.DisplayObj
oMyObj.FileName = dd.ProgramDrv  + "\\Examples\\Documents\\Austin_Drive.wmv"
oMyObj.Sound = True
```
