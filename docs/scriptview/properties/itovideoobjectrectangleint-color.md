# IToVideoObjectRectangleInt.Color

!!! abstract "Property &middot; `Scriptview.chm`"
    Property: Color for RectangleObject

Specifies the color channel of a rectangle type OverlayObject object in a video in DIAdem VIEW. If you specify colors using the ColorChannelName property, DIAdem ignores this color.

## Signature

```python
obj.Color
```
