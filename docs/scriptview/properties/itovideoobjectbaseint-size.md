# IToVideoObjectBaseInt.Size

!!! abstract "Property &middot; `Scriptview.chm`"
    Property: Size for OverlayObject

Specifies the size of an OverlayObject object in a video in DIAdem VIEW. DIAdem only takes this setting into account if you have not specified a width or height channel .

## Signature

```python
obj.Size
```
