# IToMapDisplayInt.Area

!!! abstract "Property &middot; `Scriptview.chm`"
    Property: Area for Map

Returns the area in DIAdem VIEW that contains a map display.

## Signature

```python
return_value = obj.Area
```

## Python example

```python
dd.View.Sheets(1).Areas(1).DisplayObjType = "Map"
oMyDisplayObj = dd.View.Sheets(1).Areas(1).DisplayObj
dd.MsgBoxDisp("Name " + oMyDisplayObj.Area.Name)
```
