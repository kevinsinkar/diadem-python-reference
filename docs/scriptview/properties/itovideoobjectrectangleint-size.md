# IToVideoObjectRectangleInt.Size

!!! abstract "Property &middot; `Scriptview.chm`"
    Property: Size for RectangleObject

In DIAdem VIEW, specifies the size of rectangle type OverlayObject object as a percentage of the video area. DIAdem only takes this setting into account if you have not specified a size channel.

## Signature

```python
obj.Size
```
