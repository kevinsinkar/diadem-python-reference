# IToDimensionsInt.Count

!!! abstract "Property &middot; `Scriptview.chm`"
    Property: Count for Dimensions

Returns the number of elements in a Dimensions collection in DIAdem VIEW.

## Signature

```python
obj.Count
```

## Python example

```python
dd.MsgBoxDisp("Number of Columns: " + dd.View.ActiveSheet.Cursor.Dimensions.Count)
```
