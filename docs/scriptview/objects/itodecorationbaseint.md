# IToDecorationBaseInt

!!! abstract "Object &middot; `Scriptview.chm`"
    Object: DecorationBase

The DecorationBase object provides the object for inserting a Graphic or a Text as a decoration into an axis system in DIAdem VIEW. Decorations are possible in 2D Axis Systems and also in Polar , Orbit , and Shaft Centerline axis systems .
