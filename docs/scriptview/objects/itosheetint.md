# IToSheetInt

!!! abstract "Object &middot; `Scriptview.chm`"
    Object: Sheet

The Sheet object provides access to a worksheet in DIAdem VIEW.

## Python example

```python
for oMySheet in dd.View.Sheets:
    dd.MsgBoxDisp(oMySheet.Name)
```
