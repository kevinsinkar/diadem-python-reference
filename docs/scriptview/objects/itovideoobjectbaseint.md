# IToVideoObjectBaseInt

!!! abstract "Object &middot; `Scriptview.chm`"
    Object: OverlayObject

In DIAdem VIEW, the Object object provides a type rectangle or marker OverlayObject object in a video. You use OverlayObject objects to highlight areas in a video with markers or rectangles. The OverlayObject object corresponds to one of the following objects: RectangleObject (IToVideoObjectRectangleInt) Rectangle MarkerObject (IToVideoObjectMarkerInt) Markers
