# IToCascadeCurveBaseInt

!!! abstract "Object &middot; `Scriptview.chm`"
    Object: CascadeCurveBase

The CascadeCurveBase object provides access to a Waterfall curve or an Order Line of a cascade display in DIAdem VIEW. The CascadeCurveBase object corresponds to one of the following objects: WaterfallCurve (ITo3DCurveInt) Waterfall curve Line (IToCascadeCurveInt) Order Line
