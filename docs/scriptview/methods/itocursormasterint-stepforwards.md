# IToCursorMasterInt.StepForwards

!!! abstract "Method &middot; `Scriptview.chm`"
    Method: StepForwards for Cursor

Moves the cursor forwards along the enabled curve of the enabled range in one step, or displays the next video frame of the enabled range.

## Signature

```python
obj.StepForwards()
```

## Python example

```python
oMySheet = dd.View.ActiveSheet
oMySheet.Cursor.StepForwards()
```
