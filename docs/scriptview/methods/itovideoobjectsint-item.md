# IToVideoObjectsInt.Item

!!! abstract "Method &middot; `Scriptview.chm`"
    Method: Item for OverlayObjects

Returns the OverlayObject object associated with a specific name or index in a video in DIAdem VIEW. Note You can always omit the Item method because it is the standard element of the collection.

## Signature

```python
return_value = obj.Item(NameOrIndex)
```
