# IToColorPaletteEntriesInt.Item

!!! abstract "Method &middot; `Scriptview.chm`"
    Method: Item for ColorPaletteEntries

Returns the ColorPaletteEntry object associated with a specific index in DIAdem VIEW.

## Signature

```python
return_value = obj.Item(Index)
```

## Python example

```python
MyObj = dd.View.ActiveSheet.ActiveArea.DisplayObj.UserDefinedColoringEntries
dd.MsgBoxDisp("First color: " + MyObj.Item(1).Color)
dd.MsgBoxDisp("First color: " + MyObj(1).Color)
```
