# IToTextBoxInt.Refresh

!!! abstract "Method &middot; `Scriptview.chm`"
    Method: Refresh for TextBox

Refreshes the textbox view of DIAdem VIEW. DIAdem refreshes the variable contents and the text.

## Signature

```python
obj.Refresh()
```

## Python example

```python
dd.View.Sheets(1).Areas(1).DisplayObjType = "TextBox"
dd.View.Sheets(1).Areas(1).DisplayObj.Refresh()
```
