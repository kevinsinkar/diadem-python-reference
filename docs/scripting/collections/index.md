# Collections

4 collections from `Scripting.chm`.

| Name | Summary |
| --- | --- |
| [`Dictionary`](dictionary.md) | The Dictionary object provides an array with a key/element pair. The key and the element can have any data type, but the key cannot be an array. A Dictionary object is comparable with an associative array. To use the Dictionary object, you must create this object in DIAdem with the CreateObject("Scripting.Dictionary") command. |
| [`Drives`](drives.md) | A collection of all Drive objects. You can count drives or access an individual drive using the Drives collection. |
| [`Files`](files.md) | Collection of all File objects. You can use the Files collection to count files or to access individual files. |
| [`Folders`](folders.md) | Collection of all Folder objects. You can use the Folder collection to count folders or to access individual folders. |
