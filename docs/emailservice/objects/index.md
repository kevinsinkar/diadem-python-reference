# Objects

2 objects from `EMailService.chm`.

| Name | Summary |
| --- | --- |
| [`IEMail`](iemail.md) | The EMail object provides an email to send over an email server. Use the CreateEMail method to create the EMail object. Use the Send method to send emails. Note The objects EMailService and EMail are only available in the 64-bit version of DIAdem. |
| [`IEMailService`](iemailservice.md) | The EMailService object connects to the mail server and provides the mail server with the necessary properties. Use the CreateEMailService command to create the EMailService object. The EMailService object does not support servers which demand STARTTLS. Note The objects EMailService and EMail are only available in the 64-bit version of DIAdem. |
