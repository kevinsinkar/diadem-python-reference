# Collections

6 collections from `ICANConverter.chm`.

| Name | Summary |
| --- | --- |
| [`Buses`](buses.md) | Collection of all the buses that are entered in the logfiles and that the Bus Log Converter therefore includes in the conversion. You use the buses collection to add new buses, or to delete or move buses. Every element in the buses collection is a Bus object . |
| [`DbFiles`](dbfiles.md) | Collection of all the database files that are assigned to one bus in the Bus Log Converter . Every element in the DbFiles collection is a DbFile object . |
| [`LogFiles`](logfiles.md) | Collection of all logfiles that the Bus Log Converter converts to the TDM format. You only can enter multiple logfiles in GIN-Multilogger files. Every element in the LogFiles collection is a LogFile object . |
| [`StringVector`](stringvector.md) | The StringVector object provides clusters or XNET channels in the Bus Log Converter . |
| [`UniqueStringVector`](uniquestringvector.md) | Collection of all filter texts in the Bus Log Converter . |
| [`UsedBuses`](usedbuses.md) | Collection of the buses that the Bus Log Converter includes in a filtered conversion. Every element in the UsedBuses collection is a Bus object . |
