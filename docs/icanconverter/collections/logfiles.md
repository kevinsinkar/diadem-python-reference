# LogFiles

!!! abstract "Collection &middot; `ICANConverter.chm`"
    Collection: LogFiles

Collection of all logfiles that the Bus Log Converter converts to the TDM format. You only can enter multiple logfiles in GIN-Multilogger files. Every element in the LogFiles collection is a LogFile object .

## Python example

```python
print(dd.BusLogToTDM.LogSettings.LogFiles.Count)
```
