# ICANConverter

!!! abstract "Object &middot; `ICANConverter.chm`"
    Object: BusLogToTDM

The BusLogToTDM object enables you to convert bus log files into TDM files. Refer to the page FileType for LogSettings for the supported file types.

## Python example

```python
dd.MsgBoxDisp(dd.BusLogToTDM.IsModified)
```
