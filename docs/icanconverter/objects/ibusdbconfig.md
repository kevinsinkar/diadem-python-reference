# IBusDbConfig

!!! abstract "Object &middot; `ICANConverter.chm`"
    Object: BusDbConfig

The BusDbConfig object provides a configuration for the Bus Log Converter . The BusDbConfig object contains the name and the description of the configuration, and all the buses entered in the logfile.

## Python example

```python
dd.MsgBoxDisp(dd.BusLogToTDM.BusDbConfig.Name)
```
