# ICANConverter.Clear

!!! abstract "Method &middot; `ICANConverter.chm`"
    Method: Clear for BusLogToTDM

Deletes all settings of the Bus Log Converter . For compatibility, the Clear method sets the EnumSubstitution for ResultSettings property to True. Set this property to False if you want the Bus Log Converter to generate enumeration channels .

## Signature

```python
obj.Clear()
```

## Python example

```python
dd.BusLogToTDM.Clear()
```
