# ICANConverter.Load

!!! abstract "Method &middot; `ICANConverter.chm`"
    Method: Load for BusLogToTDM

Loads a file with the extension .tcc into the Bus Log Converter . TCC files contain all the saved settings that the Bus Log Converter requires.

## Signature

```python
obj.Load(FileName)
```

## Python example

```python
dd.BusLogToTDM.Load("C:\\ConvSettings.tcc")
```
