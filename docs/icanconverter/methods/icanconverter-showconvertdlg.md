# ICANConverter.ShowConvertDlg

!!! abstract "Method &middot; `ICANConverter.chm`"
    Method: ShowConvertDlg for BusLogToTDM

Opens the Bus Log Converter dialog box, where you configure the conversion of logfiles to the TDM format.

## Signature

```python
obj.ShowConvertDlg()
```

## Notes

<div markdown="1">
<table class="Borderless" id="table1"><tr><td class="Icon"><img src="../image/note.gif"/></td><td><strong>Note</strong>  This Bus Log Converter dialog box is only available in the versions <strong>DIAdem Advanced</strong> and <strong>DIAdem Professional</strong>.</td></tr></table>
</div>

```python
dd.BusLogToTDM.ShowConvertDlg()
```
