# ICANConverter.SaveAs

!!! abstract "Method &middot; `ICANConverter.chm`"
    Method: SaveAs for BusLogToTDM

Saves all the settings of the Bus Log Converter in a file with the extension .tcc .

## Signature

```python
obj.SaveAs(FileName)
```

## Python example

```python
dd.BusLogToTDM.SaveAs("C:\\ConvSettings.tcc")
```
