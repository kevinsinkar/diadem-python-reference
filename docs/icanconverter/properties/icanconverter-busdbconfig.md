# ICANConverter.BusDbConfig

!!! abstract "Property &middot; `ICANConverter.chm`"
    Property: BusDbConfig for BusLogToTDM

Specifies the configuration of buses and the associated database files in the Bus Log Converter . You cannot overwrite the BusDbConfig property.

