# IResultSettings.EnumSubstitution

!!! abstract "Property &middot; `ICANConverter.chm`"
    Property: EnumSubstitution for ResultSettings

Specifies whether the Bus Log Converter replaces enumeration strings of a logfile that is to be converted in the data channel of the TDM file, by NoValues or generates enumeration channels .

## Signature

```python
obj.EnumSubstitution
```

## Python example

```python
dd.BusLogToTDM.ResultSettings.EnumSubstitution = False
```
