# IResultSettings.ResultFile

!!! abstract "Property &middot; `ICANConverter.chm`"
    Property: ResultFile for ResultSettings

Specifies the name and the path of the TDM file into which the CAN Converter converts the selected logfiles in the Bus Log Converter .

## Signature

```python
obj.ResultFile
```

## Python example

```python
dd.BusLogToTDM.ResultSettings.ResultFile = "C:\\CANData\\Results\\CANConverter.tdm"
```
