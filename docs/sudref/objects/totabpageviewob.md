# ToTabPageViewOb

!!! abstract "Object &middot; `Sudref.chm`"
    Object: Page

The Page object corresponds to a tab in a user dialog box. You use the TabPageCtrl object to define a series of tabs. Each tab can contain any number of controls.

## Python example

!!! warning "Machine-translated"
    The original DIAdem topic did not include a Python tab; this
    example was machine-translated from the VBScript source.

```python
Page1.Visible = 0  # 0=No, 1=Yes
```
