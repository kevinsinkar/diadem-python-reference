# Text

!!! abstract "Object &middot; `Sudref.chm`"
    Object: Text <XTable>

The Text object corresponds to the Text control in an extended table of a user dialog box. You use the Text object to define a text to be displayed in an extended table. The text can include DIAdem expressions . Note You only can access this control within the event procedures of the extended table.
