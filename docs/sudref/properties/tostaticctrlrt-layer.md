# ToStaticCtrlRt.layer

!!! abstract "Property &middot; `Sudref.chm`"
    Property: Layer for Text

Specifies in user dialog boxes the layer in which a control is located. Use this property to make the controls you assign to one layer visible or invisible while you generate or display the user dialog box. The property does not affect which control is on top.

</tr>
<tr>
<td class="Icon"><img src="../image/note.gif"/></td>
<td><strong>Note  </strong>A control that is located in a tab is automatically assigned to the layer of the <span class="Monospace">TabPageCtrl</span> control.</td>
