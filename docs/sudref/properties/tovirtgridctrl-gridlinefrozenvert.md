# ToVirtGridCtrl.gridlinefrozenvert

!!! abstract "Property &middot; `Sudref.chm`"
    Property: GridLineFrozenVert for XTable

Specifies in user dialog boxes whether a thicker line separates the frozen columns of an extended table from the movable columns. The property can have the following settings: 0 - No Does not separate the frozen columns. 1 - Yes A line separates the frozen columns.

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img src="../image/note.gif"/></td><td><strong>Note</strong>  The user dialog box only displays the separating line when the value of the <a href="../tovirtgridctrl-columnsfrozen/">ColumnsFrozen</a> property is greater than 0.</td></tr></table>
