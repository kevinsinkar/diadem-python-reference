# ToVarListBoxRt.horizontalscroll

!!! abstract "Property &middot; `Sudref.chm`"
    Property: HorizontalScroll for EnumListBox

Specifies in user dialog boxes whether a selection list displays a horizontal scroll bar. The property can have the following settings: 0 - None Does not display a scroll bar. 1 - Always Always displays a scroll bar.
