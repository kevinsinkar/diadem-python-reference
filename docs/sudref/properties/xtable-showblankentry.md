# XTable.ShowBlankEntry

!!! abstract "Property &middot; `Sudref.chm`"
    Property: ShowBlankEntry for ChnComboBox <XTable>

Specifies whether the channel selection list of an extended table displays a blank entry as the first entry. The property can have the following settings: 0 - No Does not display the blank entry. 1 - Yes Displays the blank entry.

## Notes

<div markdown="1">
<table class="Borderless">
<tr>
<td class="Icon"><img src="../image/note.gif"/></td>
<td><strong>Note</strong>  If the channel selection list contains a blank entry the indexing for <a href="../xtable-selection/">Selection</a> and <a href="../xtable-topindex/">TopIndex</a>starts at <span class="Monospace">0</span> instead of <span class="Monospace">1</span>. If you select the blank entry, <a href="../xtable-value-2/">Value</a> receives the value <span class="Monospace">0</span> instead of the channel number.</td>
</tr>
</table>
</div>
