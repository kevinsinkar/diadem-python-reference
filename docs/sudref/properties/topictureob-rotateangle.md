# ToPictureOb.RotateAngle

!!! abstract "Property &middot; `Sudref.chm`"
    Property: RotateAngle for Picture

Specifies the rotation angle of a control in a user dialog box. Specify the rotation angle in degrees. Positive values rotate the control clockwise, negative values rotate the control counter-clockwise.

## Signature

```python
obj.RotateAngle
```
