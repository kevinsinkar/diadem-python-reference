# XTable.InterpretedTextEnabled

!!! abstract "Property &middot; `Sudref.chm`"
    Property: InterpretedTextEnabled for ListBox <XTable>

Specifies in a user dialog box whether text from the extended table which contains DIAdem expressions is interpreted. DIAdem expressions are DIAdem variables or formula expressions enclosed in @ , or script expressions from registered DIAdem scripts, enclosed in @@ . Note If the contents of a control which is connected to a DIAdem variable changes while the user dialog box is active, you must refresh the associated control with the Refresh method. The property can have the following settings: 0 - No Does not interpret DIAdem expressions. 1 - Yes Interprets DIAdem expressions.
