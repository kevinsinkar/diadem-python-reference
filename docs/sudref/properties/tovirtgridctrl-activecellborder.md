# ToVirtGridCtrl.activecellborder

!!! abstract "Property &middot; `Sudref.chm`"
    Property: ActiveCellBorder for XTable

Specifies in user dialog boxes whether the enabled cell in an extended table has a frame. The property can have the following settings: 0 - Always The enabled cell is framed. 1 - Never The enabled cell is not framed. 2 - IfActive The enabled cell is only framed if the focus is on the extended table.
