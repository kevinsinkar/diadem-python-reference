# ToTabPageCtrl.Pages

!!! abstract "Property &middot; `Sudref.chm`"
    Property: Pages for TabPageCtrl

Returns a collection of all the tabs of a TabPageCtrl control in user dialog boxes. To access single tabs in scripts, either use the Item property or enter the index or the name in parentheses.

## Signature

```python
return_value = obj.Pages
```
