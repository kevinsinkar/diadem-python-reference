# ToTreeNodeInt.ForeColor

!!! abstract "Property &middot; `Sudref.chm`"
    Property: ForeColor for TreeNode

Specifies in user dialog boxes the text color of a tree node. If you do not use the property in the events EventInitialize or EventRefresh , DIAdem does not automatically show the changes in the tree. Call the Update method to refresh a node after making changes to this node.

## Signature

```python
obj.ForeColor
```
