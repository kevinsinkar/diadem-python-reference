# ToSudViewOb.Left

!!! abstract "Property &middot; `Sudref.chm`"
    Property: Left for Dialog

Specifies the distance from the left edge of a user dialog box to the screen origin. The screen origin is the top-left corner of the screen. The right edge of the user dialog box remains unchanged. To move a dialog box, use the Move method.

## Signature

```python
obj.Left
```

## Python example

!!! warning "Machine-translated"
    The original DIAdem topic did not include a Python tab; this
    example was machine-translated from the VBScript source.

```python
Dialog.Left = 50
```
