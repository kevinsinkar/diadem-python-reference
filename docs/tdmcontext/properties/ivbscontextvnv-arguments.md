# IVbsContextVnV.Arguments

!!! abstract "Property &middot; `TDMcontext.chm`"
    Property: Arguments for ContextVnV <Data Preprocessor>

Returns the Arguments <SystemLink> collection in the On_ValidationAndVerification procedure. This collection contains some arguments transferred by the system, such as path and processed file. You can read and overwrite the elements of the collection and add your own elements to the collection.

## Signature

```python
return_value = obj.Arguments
```
