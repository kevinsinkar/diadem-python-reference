# DCGenFileName

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: DCGenFileName

Specifies whether DIAdem automatically creates the name of the file where DIAdem saves the measured values. The created name consists of the base filename and the identifiers date, time, and/or sequential number. If this variable is TRUE , DIAdem appends these identifiers depending on DCGenFileNameDate , DCGenFileNameTime , and DCGenFileNameId .
