# XNETTermination

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: XNETTermination

Specifies whether to enable the bus termination for CAN or for LIN on the hardware. If the value is TRUE , DIAdem enables the bus termination on the hardware.
