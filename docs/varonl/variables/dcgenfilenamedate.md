# DCGenFileNameDate

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: DCGenFileNameDate

Specifies whether DIAdem appends the date to the base name . If the identifier is TRUE and DCGenFileName is set, DIAdem appends these values.
