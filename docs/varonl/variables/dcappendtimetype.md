# DCAppendTimeType

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: DCAppendTimeType

Specifies whether DIAdem continues the time axis of the DAT file without gaps or whether DIAdem uses the current measurement time for the measurement values to be appended.
