# DCChnXYRelationHandling

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: DCChnXYRelationHandling

Specifies whether DIAdem creates XY-channel references between the x-channel and the y-channels when saving. DIAdem interprets the first channel as the x-channel. When you select the “UseAppSettings” setting, DIAdem uses the DIAdem Settings .
