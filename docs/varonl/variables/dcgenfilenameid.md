# DCGenFileNameId

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: DCGenFileNameId

Specifies whether DIAdem appends an index to the base name . If the identifier is TRUE and DCGenFileName is set, DIAdem appends these values.
