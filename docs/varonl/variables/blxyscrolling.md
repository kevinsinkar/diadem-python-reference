# BlXYScrolling

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: BlXYScrolling

Specifies whether DIAdem automatically adjusts the scale range of the x-axis and of the y-axis to the signal and moves the axes of an xy-display like a recorder.
