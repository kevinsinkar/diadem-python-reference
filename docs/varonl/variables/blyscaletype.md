# BlYScaleType

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: BlYScaleType

Specifies the display mode for the y-axis. You can choose between one y-axis and several y-axes, and between a display that is a percentage of the display area and a display of the physical values.
