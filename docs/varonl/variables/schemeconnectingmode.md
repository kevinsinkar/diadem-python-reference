# SchemeConnectingMode

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: SchemeConnectingMode

Specifies whether DIAdem DAC assigns the signals to be connected to the terminals of a block in the name-oriented or in the number-oriented mode. The default value is ByNumber which specifies that DIAdem connects signals in the number-oriented mode.
