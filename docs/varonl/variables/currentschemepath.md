# CurrentSchemePath

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: CurrentSchemePath

Contains the path of the currently loaded block diagram.

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img src="image/note.gif"/></td><td><strong>Note  </strong>If either DIAdem DAC or DIAdem VISUAL is not yet open, the variable <span class="Monospace">CurrentSchemePath</span> contains an empty string.</td></tr></table>
<table class="Borderless"><tr><td class="Icon"><img src="image/note.gif"/></td><td><strong>Note  </strong>Use the variable <a href="../currentschemefilename/">CurrentSchemeFilename</a> to receive the name and path of the currently loaded block diagram. Use the <a href="../actschemename/">ActSchemeName</a> variable to get the name of the loaded block diagram or the open subblock diagram without a path and without a filename extension.</td></tr></table>
</div>
