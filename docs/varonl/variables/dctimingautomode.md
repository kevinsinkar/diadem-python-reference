# DCTimingAutoMode

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: DCTimingAutoMode

Specifies how DIAdem determines the automatic Timing selection. If you use the Default value, DIAdem sets the Timing selection to the method that is the most suitable for the DIAdem version. Otherwise DIAdem uses the method that matches the specified DIAdem version.
