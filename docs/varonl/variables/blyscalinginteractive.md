# BlYScalingInteractive

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: BlYScalingInteractive

Specifies whether you can change the scale of the display manually during a measurement. Refer to the Help page Manual Axis Scaling for further information.
