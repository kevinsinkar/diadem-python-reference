# DCGenFileNameTime

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: DCGenFileNameTime

Specifies whether DIAdem appends the time to the base name . If the identifier is TRUE and DCGenFileName is set, DIAdem appends these values.
