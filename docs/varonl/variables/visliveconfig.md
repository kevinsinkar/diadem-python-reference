# VisLiveConfig

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: VisLiveConfig

Specifies whether you can change the settings for VISUAL displays during the measurement. The default value TRUE indicates that you can change the VISUAL settings. Disable the VISUAL display changes for time critical measurements. DIAdem saves this information with the block diagram.
