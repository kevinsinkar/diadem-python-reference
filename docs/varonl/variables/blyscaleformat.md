# BlYScaleFormat

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: BlYScaleFormat

Specifies the format specification of the y-scales. By specifying a format, you can prevent DIAdem from truncating the scales following changes to the value range. If you do not enter a value, DIAdem automatically specifies the scales above the given value range.
