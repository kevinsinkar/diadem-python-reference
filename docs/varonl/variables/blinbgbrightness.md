# BlInBgBrightness

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: BlInBgBrightness

Specifies gray as the second color of a color shading that DIAdem displays on a push button when the button is pressed. A brightness value of 0 is black, and a value of 255 is white.
