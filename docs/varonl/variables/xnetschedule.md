# XNETSchedule

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: XNETSchedule

Specifies the name of the LIN schedule if the network node is working as a LIN master. An empty string indicates that the network node works as a LIN slave. In the Local Interconnect Network, the master requests the data from the slaves.
