# XNETListenOnly

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: XNETListenOnly

Specifies whether the CAN interface only participates passively in the network protocol. If the value is TRUE , the driver only monitors the network and the XNET driver cannot output data.
