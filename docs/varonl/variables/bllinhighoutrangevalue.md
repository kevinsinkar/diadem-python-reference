# BlLinHighOutRangeValue

!!! abstract "Variable &middot; `VarOnl.chm`"
    Variable: BlLinHighOutRangeValue

Specifies the value which DIAdem uses in Free Linearization through Table and Free Linearization through Channels for the result data when the value range of the output values is exceeded.
