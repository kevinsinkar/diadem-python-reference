# Sheets

!!! abstract "Collection &middot; `ReportApi.chm`"
    Collection: Sheets

Collection of all Sheet objects in DIAdem REPORT. Use the Sheets collection to access individual worksheets in DIAdem-REPORT.

## Python example

```python
for oMySheet in dd.Report.Sheets:
    dd.MsgBoxDisp(oMySheet.Name)
```
