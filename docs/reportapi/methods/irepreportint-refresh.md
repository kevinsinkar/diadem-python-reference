# IRepReportInt.Refresh

!!! abstract "Method &middot; `ReportApi.chm`"
    Method: Refresh for Report

Refreshes the view in DIAdem REPORT. Use the RefreshSilent method to plot the layout in the background and to then display it when the layout is completed. If the property Enable for CurveExpansionSettings is True , DIAdem automatically expands the layout, for which the curve expansion is active, before the refresh.

## Signature

```python
obj.Refresh()
```

## Python example

```python
dd.Report.Refresh()
```
