# IRepReportInt.RefreshSilent

!!! abstract "Method &middot; `ReportApi.chm`"
    Method: RefreshSilent for Report

Refreshes the view in DIAdem-REPORT by plotting the layout in the background and making it visible once the layout has been plotted. Use the Refresh method to plot the layout in the foreground. If the property Enable for CurveExpansionSettings is True , DIAdem automatically expands the layout, for which the curve expansion is active, before the refresh.

## Signature

```python
obj.RefreshSilent()
```

## Python example

```python
dd.Report.RefreshSilent()
```
