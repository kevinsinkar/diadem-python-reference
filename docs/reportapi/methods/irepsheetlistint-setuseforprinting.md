# IRepSheetListInt.SetUseForPrinting

!!! abstract "Method &middot; `ReportApi.chm`"
    Method: SetUseForPrinting for Sheets

Specifies that DIAdem includes the worksheets when printing. Use the UseForPrinting for Sheet property to specify whether DIAdem includes a worksheet when printing.

## Signature

```python
obj.SetUseForPrinting(UseForPrinting)
```

## Python example

```python
dd.Report.Sheets.SetUseForPrinting(True)
```
