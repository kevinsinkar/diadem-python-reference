# IRepReportInt.SaveLayout

!!! abstract "Method &middot; `ReportApi.chm`"
    Method: SaveLayout for Report

Saves the current REPORT layout under the specified name. Layout files have the filename extension *.tdr .

## Signature

```python
obj.SaveLayout(FileName)
```

## Python example

```python
dd.Report.SaveLayout("MyLayout.tdr")
```
