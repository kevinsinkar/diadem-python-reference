# IRepSettingsInt.Curve2DTransformationConfiguration

!!! abstract "Property &middot; `ReportApi.chm`"
    Property: Curve2DTransformationConfiguration for Settings

Specifies the command for the curve transformation in a 2D axis system in DIAdem REPORT. Specify this command in the Curve and Axis Definition dialog box when you click the Enable Curve Transformation button and then the Curve Transformation button.

## Signature

```python
obj.Curve2DTransformationConfiguration
```

## Python example

```python
oMySettings = dd.Report.Settings
oMySettings.Curve2DTransformationConfiguration = "UserCmdExample"
```
