# IRepPaletteLineColorInt.RGBFilling

!!! abstract "Property &middot; `ReportApi.chm`"
    Property: RGBFilling for PaletteColor

DIAdem currently does not support this property. Specifies the RGB values of the second background color of a color shading in DIAdem REPORT. First assign the value eColorIndexFillEffects to the property ColorIndex . The RGB property specifies the first color of a filling effect.
