# IRepSettingsInt.Curve3DTransformationConfiguration

!!! abstract "Property &middot; `ReportApi.chm`"
    Property: Curve3DTransformationConfiguration for Settings

Specifies the command for the curve transformation in a 3D axis system in DIAdem REPORT. Specify this command in the Curve and Axis Definition dialog box when you click the Enable Curve Transformation button and then the Curve Transformation button.

## Signature

```python
obj.Curve3DTransformationConfiguration
```

## Python example

```python
oMySettings = dd.Report.Settings
oMySettings.Curve3DTransformationConfiguration = "UserCmdExample"
```
