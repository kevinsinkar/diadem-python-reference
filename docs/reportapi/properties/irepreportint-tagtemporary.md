# IRepReportInt.TagTemporary

!!! abstract "Property &middot; `ReportApi.chm`"
    Property: TagTemporary for Report

Specifies any temporary label for a layout in DIAdem REPORT. Use this property to temporarily add additional information to the layout. Afterwards you can evaluate such information in a program, for example, for special treatment of the layout. The tag does not affect the behavior of the layout. DIAdem does not save this tag with the layout.

## Signature

```python
obj.TagTemporary
```

## Python example

```python
dd.Report.TagTemporary = "Selected"
```
