# IRepSettingsInt.UseVirtualPrintMemory

!!! abstract "Property &middot; `ReportApi.chm`"
    Property: UseVirtualPrintMemory for Settings

Specifies whether DIAdem REPORT manages the temporary printer data in the memory or whether DIAdem outsources the data to the hard drive, when printing and exporting a REPORT layout. DIAdem saves this information in the desktop file.

## Signature

```python
obj.UseVirtualPrintMemory
```

## Python example

```python
dd.Report.Settings.UseVirtualPrintMemory = True
```
