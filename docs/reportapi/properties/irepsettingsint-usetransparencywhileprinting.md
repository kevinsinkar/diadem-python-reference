# IRepSettingsInt.UseTransparencyWhilePrinting

!!! abstract "Property &middot; `ReportApi.chm`"
    Property: UseTransparencyWhilePrinting for Settings

Specifies whether DIAdem REPORT includes the transparency of the color when printing a layout. Some printers cannot output transparency and instead print transparent areas without color.

## Signature

```python
obj.UseTransparencyWhilePrinting
```

## Python example

```python
dd.Report.Settings.UseTransparencyWhilePrinting = False
```
