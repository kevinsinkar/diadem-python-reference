# IRepBoxWhiskerMarkerFillingInt.GradientDirection

!!! abstract "Property &middot; `ReportApi.chm`"
    Property: GradientDirection for 2DBoxWhiskerMarkerFilling

DIAdem does not support this property at the moment. Specifies the color shading direction of the curve marker filling of a 2D axis system in the Box Whisker display mode in DIAdem REPORT. You also can use the SetFillingColor method to specify filling effects. Not all REPORT objects display filling effects.

```python
obj.GradientDirection
```
