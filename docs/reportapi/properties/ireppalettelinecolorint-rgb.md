# IRepPaletteLineColorInt.RGB

!!! abstract "Property &middot; `ReportApi.chm`"
    Property: RGB for PaletteColor

DIAdem currently does not support this property. Specifies the RGB values of a color in DIAdem REPORT. First assign the value eColorIndexOtherColors to the property ColorIndex . If you assign the value eColorIndexFillEffects to the property ColorIndex , the RGB property specifies the first color of a filling effect. Not all REPORT objects support filling effects.
