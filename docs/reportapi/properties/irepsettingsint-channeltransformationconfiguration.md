# IRepSettingsInt.ChannelTransformationConfiguration

!!! abstract "Property &middot; `ReportApi.chm`"
    Property: ChannelTransformationConfiguration for Settings

Specifies the command for the curve transformation in a 2D table in DIAdem REPORT. Specify this command in the Table Definition dialog box by clicking the Enable Channel Transformation button and then the Channel Transformation button.

## Signature

```python
obj.ChannelTransformationConfiguration
```

## Python example

```python
oMySettings = dd.Report.Settings
oMySettings.ChannelTransformationConfiguration = "UserCmdExample"
```
