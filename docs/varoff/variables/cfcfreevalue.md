# CFCFreeValue

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: CFCFreeValue

Specifies the filter class when the CFC filtering is non-standardized. If you do not specify the CFCFreeValue variable, DIAdem uses the value 0 .
