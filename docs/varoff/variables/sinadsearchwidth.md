# SINADSearchWidth

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: SINADSearchWidth

Specifies the relative frequency width to search for the fundamental frequency in the frequency domain. Specify the frequency width as a percentage of the sampling rate.
