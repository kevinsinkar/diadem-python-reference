# ReportDefFont

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ReportDefFont

Specifies the character set for all texts and numbers in a report if no character set is specified.

## Python example

```python
dd.ReportDefFont="Times Roman"()
PicUpdate()
```
