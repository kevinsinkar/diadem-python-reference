# AudioAutoRepeat

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: AudioAutoRepeat

Specifies whether DIAdem restarts the audio output automatically at the end. If the value is TRUE , DIAdem starts the audio output automatically.
