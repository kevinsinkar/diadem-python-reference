# FileDlgPathKey

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: FileDlgPathKey

Specifies whether DIAdem stores the names of the selected files with absolute paths or without paths in the ASCII file. The default value is FALSE which specifies that DIAdem does not save the absolute path as well.
