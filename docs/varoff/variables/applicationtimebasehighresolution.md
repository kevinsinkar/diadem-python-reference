# ApplicationTimebaseHighResolution

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ApplicationTimebaseHighResolution

Specifies whether DIAdem creates time channels with a high resolution. Specify this variable before loading data and before assigning variables, because time values in channels and variables otherwise return false data.
