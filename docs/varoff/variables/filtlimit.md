# FiltLimit

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: FiltLimit

Specifies the 3 dB limit frequency of an IIR filter for highpass and lowpass. For allpass of an FIR filter, it specifies the frequency where the phase angle rotation reaches half the phase angle rotation of the overall system.
