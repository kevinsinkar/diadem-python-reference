# ApplicationLegacyBinaryOperations

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ApplicationLegacyBinaryOperations

Specifies whether bit operations get the same result as in DIAdem versions 2017, 2018, and 2019. In these versions, DIAdem incorrectly included the highest bit in some bit operations. To obtain the same result for bit operations as in DIAdem versions 2017, 2018, and 2019, you must assign the value True to the ApplicationLegacyBinaryOperations variable. DIAdem does not save the ApplicationLegacyBinaryOperations variable of the desktop file.
