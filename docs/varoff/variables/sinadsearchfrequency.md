# SINADSearchFrequency

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: SINADSearchFrequency

Specifies the approximate center frequency which DIAdem uses to search for the fundamental frequency in the frequency domain. If the value is -1 , DIAdem uses the frequency with the greatest amplitude as fundamental frequency.
