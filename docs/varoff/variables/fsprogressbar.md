# FSProgressbar

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: FSProgressbar

Specifies whether the progress bar remains visible during long operations. If you assign the value TRUE to the FSProgressBar variable, DIAdem does not display the progress bar. The default value for the FSProgressBar variable is FALSE .
