# ViewSoftwareRendering

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ViewSoftwareRendering

Specifies whether DIAdem uses the acceleration capabilities of the video card. If the value is TRUE , the graphics acceleration of the graphics card is disabled. Only disable graphics acceleration if problems with the display occur in 3D models.
