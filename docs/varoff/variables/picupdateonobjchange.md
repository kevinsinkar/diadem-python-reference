# PicUpdateOnObjChange

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: PicUpdateOnObjChange

Specifies whether DIAdem replots the entire worksheet after you have changed individual objects. DIAdem saves this information in the desktop file.
