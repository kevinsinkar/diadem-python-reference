# OrdATOSum

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: OrdATOSum

Specifies whether DIAdem calculates the sum level of the amplitude channel. The sum level corresponds with the floating square mean over all frequencies of the signal.
