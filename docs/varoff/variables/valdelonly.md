# ValDelOnly

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ValDelOnly

Specifies whether DIAdem deletes only the channel values or also all properties when deleting all values. The default value is FALSE which specifies that DIAdem also deletes the properties.
