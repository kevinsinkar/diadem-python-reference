# UsePropTemplate

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: UsePropTemplate

Specifies whether you define custom properties as a template for a data set, for channel groups, and for channels. When you load new data into the Data Portal or when you create new channel groups or channels, DIAdem adds the custom properties from this template to the data set, to the channel groups, and to the channels respectively. DIAdem generates the custom properties of a template in the data set as soon as you delete the internal data store.
