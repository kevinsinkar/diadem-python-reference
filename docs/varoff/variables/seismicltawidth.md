# SeismicLTAWidth

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: SeismicLTAWidth

Specifies the time span for calculating the long-term average (LTA) in seconds. The LTA corresponds with the background noise of a seismic signal.
