# ChnCommentOver

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ChnCommentOver

Specifies whether DIAdem overwrites existing channel comments with calculation operations.

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img height="25" src="image/note.gif" width="26"/></td><td><strong>Note  </strong>All DIAdem functions that generate new channels also automatically generate new channel comments. DIAdem saves this variable in the desktop file. In the shipped desktop file, the <span class="Monospace">ChnCommentOver</span> variable has the value <span class="Monospace">True</span>, so that DIAdem overwrites existing comments.</td></tr></table>
</div>
