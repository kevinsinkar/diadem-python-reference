# UseChnXYRelationHandling

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: UseChnXYRelationHandling

Specifies that DIAdem uses channel references in xy-channels and whether the x-channel is used when copying and moving xy-channels. The default value is TRUE .
