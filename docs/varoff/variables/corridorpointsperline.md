# CorridorPointsPerLine

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: CorridorPointsPerLine

Specifies the number of extra partitions between two points of the input signal. A greater number of partitions creates curves that adjust better to the signal, especially with greater distances, but this also needs more computing time.
