# MsgNotModal

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: MsgNotModal

Specifies whether DIAdem opens a message box modally or non-modally. The default value is FALSE which means that DIAdem opens the message box in modal mode.

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img height="25" src="image/note.gif" width="26"/></td><td><strong>Note</strong>  If you use the <a href="../../../comoff/commands/msgboxdisp/">MsgBoxDisp</a> command to open a message box and if the <a href="../msgtype/">MsgType</a> variable has the value <span class="Monospace">MsgTypeWarning</span>, DIAdem ignores the value of the <span class="Monospace">MsgNotModal</span> variable and always opens the message box modally.</td></tr></table>
</div>
