# MqttUrl

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: MqttUrl

Specifies the URL of the MQTT broker. The URL can also contain the port. By default, DIAdem uses port 1883 for unencrypted communication and port 8883 for encrypted communication.
