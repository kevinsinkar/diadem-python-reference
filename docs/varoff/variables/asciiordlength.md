# ASCIIOrdLength

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ASCIIOrdLength

Specifies the minimum length for ordinal numbers in ASCII block storage.

## Python example

```python
dd.ASCIIOrdLength = 5
MyNumber = 100 #  100'
MyNumber = 9 #    9'
dd.ASCIIOrdLength = 2
MyNumber = 103 #103'
```
