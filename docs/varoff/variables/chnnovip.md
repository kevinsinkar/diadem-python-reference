# ChnNovIP

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ChnNovIP

Specifies whether DIAdem overwrites the values of the input channels with the result channels of NoValue processing. The default value is FALSE , which specifies that DIAdem does not overwrite the input channels.
