# PrintColor

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: PrintColor

Specifies whether DIAdem prints in color, in black and white, or in the color setting configured on the printer.

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img src="image/note.gif"/></td><td><strong>Note  </strong>The color setting is not supported by all printers. Depending on the printer, it might print a color print even if you set <span class="Monospace">Monochrome</span>.</td></tr></table>
</div>
