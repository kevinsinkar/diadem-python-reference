# AudioBuffersize

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: AudioBuffersize

Specifies the number of values DIAdem uses for the buffer of the audio output. If you do not specify this value, DIAdem specifies the size of the buffer automatically.
