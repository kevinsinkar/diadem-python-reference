# StatsUsePopulationFormula

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: StatsUsePopulationFormula

Specifies whether DIAdem calculates the characteristic values based on the population or on the sample. If the value is False , DIAdem calculates the characteristic values based on samples.
