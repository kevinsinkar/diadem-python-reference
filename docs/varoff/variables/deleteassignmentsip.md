# DeleteAssignmentsIP

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: DeleteAssignmentsIP

Specifies whether DIAdem replaces the assignment channel with the numeric channel. The default value of the variable is FALSE , so that DIAdem retains the assignment channel and creates an additional numeric channel.
