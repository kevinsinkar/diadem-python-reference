# GaussFitResidue

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: GaussFitResidue

Receives the residue or the weighted mean error of a Gaussian curve fitting. If you select the Least absolute residual algorithm, the variable receives the absolute error of the weighted mean. If you use other algorithms, the variable does not receive an absolute value.
