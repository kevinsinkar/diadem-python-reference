# GaussFitTolerance

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: GaussFitTolerance

Specifies the tolerance that specifies when to end the iterative fit of the amplitude, the center, and the standard deviation in the bisquare method for a Gaussian curve fitting. If the relative difference of the residual undershoots the tolerance in two successive cycles, the curve fitting stops. If the tolerance is 0, DIAdem sets the tolerance to 0.0001.
