# PDFAppend

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: PDFAppend

Specifies whether DIAdem appends a PDF export to an existing PDF file. The default value is FALSE and specifies that DIAdem does not append the PDF export.
