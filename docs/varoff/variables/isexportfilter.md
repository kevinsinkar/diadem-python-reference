# IsExportFilter

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: IsExportFilter

Specifies which file filter DIAdem checks. If you assign the value TRUE to the IsExportFilter variable, or if you do not specify the variable, DIAdem checks for the import filter otherwise for the export filter.
