# PropInheritanceMode

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: PropInheritanceMode

Specifies whether DIAdem transfers properties of elements from superordinate levels when loading an element from a ASAM data store to the element in the Data Portal. DIAdem only uses the settings you make here if you select Same as NAVIGATOR Settings in the Edit Loading Configuration dialog box.
