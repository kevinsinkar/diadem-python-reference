# ReplaceWithNoValue

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ReplaceWithNoValue

Specifies that DIAdem replaces the values in the numeric channel with NoValues, which are connected to assignments in the assignment channel. The default value of the variable is FALSE , so that DIAdem does not replace the values originally connected with assignments with NoValues.
