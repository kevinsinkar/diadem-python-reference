# D2TabRow

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: D2TabRow

Specifies the row index of the 2D table.

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img height="25" src="image/note.gif" width="26"/></td><td><strong>Note  </strong>The index of the first table row is always 1. If you want to start listing data from the 8th row of a 2D table, add the number of rows to be skipped to the <span class="Monospace">D2TabRow</span> variable, for example, <span class="Monospace">D2TabRow+7</span>.</td></tr></table>
</div>
