# MapLinInnerVal

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: MapLinInnerVal

Specifies the value that DIAdem uses in the linear mapping process of spike curves for those values to which DIAdem does not assign an interpolation point. The default value of the MapLinInnerVal variable is 0 .
