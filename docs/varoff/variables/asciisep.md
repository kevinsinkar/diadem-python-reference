# ASCIISep

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ASCIISep

Specifies the delimiter for ASCII block storage. If the data is stored blockwise, DIAdem generates a row in the ASCII file for each data block, and separates the single values with the delimiter.
