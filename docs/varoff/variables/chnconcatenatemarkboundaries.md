# ChnConcatenateMarkBoundaries

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ChnConcatenateMarkBoundaries

Specifies that DIAdem inserts a separator after each channel to make the original channels visible. DIAdem inserts NoValue for numeric channels and the following character: .---.-. for text channels.
