# SRTMTilesCacheFolder

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: SRTMTilesCacheFolder

Specifies the folder where DIAdem buffers the SRTM data. The path is based on the path you defined in the variable SharedDataPath . If you change this path, you must download all data again.
