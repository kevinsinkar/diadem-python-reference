# XmsNegSigValCalc

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: XmsNegSigValCalc

Specifies whether DIAdem includes the negative signal sections in the X ms calculation. The default value is FALSE which specifies that DIAdem does not consider the negative signal parts.
