# PulseDetPulseWidth

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: PulseDetPulseWidth

Specifies the minimum width of a pulse. The default value is 2 , which means that two consecutive tachometer values must reach the threshold for a pulse to be detected.
