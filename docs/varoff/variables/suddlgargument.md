# SUDDlgArgument

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: SUDDlgArgument

Specifies the transfer parameter for a user dialog box. In the SUD editor you use the GetArgument method to access the transfer parameters. The default value of the SUDDlgArgument variable is NULL .
