# ResampleCorrelationMode

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ResampleCorrelationMode

Specifies whether DIAdem runs the cross correlation calculation in the frequency domain or the time domain. The default value of the variable is “FrequencyDomain” , so that DIAdem executes the calculation in the frequency domain.
