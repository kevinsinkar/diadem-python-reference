# ExcelFileName

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ExcelFileName

Specifies the Excel file that DIAdem opens. If you do not specify an Excel file, the command creates an Excel workbook with an empty worksheet named Sheet1 .
