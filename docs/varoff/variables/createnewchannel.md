# CreateNewChannel

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: CreateNewChannel

Specifies whether DIAdem saves the result in a new channel ( TRUE ) or whether DIAdem overwrites the input channel with the result ( FALSE ).
