# PDFOptimization

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: PDFOptimization

Specifies whether DIAdem optimizes PDF files for fast web display. To do so, DIAdem restructures a PDF document in such a way that it can be downloaded page-wise from a web server. DIAdem does not optimize PDF files by default. If the value is TRUE , DIAdem optimizes PDF files.
