# MultipleSortExtOrder

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: MultipleSortExtOrder

Specifies whether DIAdem sorts values and text in ascending or descending order. DIAdem sorts texts according to the ASCII table.
