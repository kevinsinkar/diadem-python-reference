# ValOnly

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ValOnly

Specifies whether DIAdem should overwrite the channel names, the description, and the unit, when the target channel is length zero and the clipboard contains the channel name, the description, and the unit. For example, the DataBlClpCopy command copies the channel name and the unit to the clipboard. The default value is FALSE which specifies that DIAdem overwrites the properties.
