# ReportPrintTransparency

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ReportPrintTransparency

Specifies whether DIAdem includes the transparency of the color when printing. Some printers cannot output transparency and instead print these areas without color. If you disable this setting, DIAdem prints transparent colors opaque without transparency.
