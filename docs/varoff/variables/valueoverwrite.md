# ValueOverwrite

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ValueOverwrite

Specifies whether DIAdem overwrites existing channel data. The default value is FALSE which specifies that DIAdem does not overwrite the input channels.
