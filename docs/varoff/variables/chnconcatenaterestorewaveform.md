# ChnConcatenateRestoreWaveform

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ChnConcatenateRestoreWaveform

Specifies that DIAdem saves the result channels like the input channels as waveform channels if possible. This requires the new time axis of each waveform channel to be continually equidistant.
