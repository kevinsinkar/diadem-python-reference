# XOffsetDeltaTs

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: XOffsetDeltaTs

Receives in an offset calculation the absolute time values of the maximum offset of the second xy-curve to the first xy-curve. If the variable receives, for example, the value 4 , you must move the second xy-curve by -4 in the x-direction so that the y-values correspond.
