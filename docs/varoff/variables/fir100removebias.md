# Fir100RemoveBias

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: Fir100RemoveBias

Specifies whether DIAdem eliminates the bias in FIR100 filtering. The default value is FALSE , which specifies that DIAdem does not remove the bias.
