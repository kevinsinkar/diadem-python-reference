# ClassDontUseNV

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ClassDontUseNV

Specifies whether DIAdem ignores or replaces NoValues in a reducing classification. The default value is FALSE , which specifies that DIAdem does not replace NoValues.
