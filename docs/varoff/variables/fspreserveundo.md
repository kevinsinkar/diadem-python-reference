# FSPreserveUndo

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: FSPreserveUndo

Specifies whether DIAdem moves files and folders into the recycle bin or deletes them irretrievably. The default value is FALSE which specifies that DIAdem deletes the files irrevocably.
