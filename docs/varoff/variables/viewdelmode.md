# ViewDelMode

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ViewDelMode

Specifies whether DIAdem deletes the data or replaces the data with NoValues when you select Remove Data Points on the DIAdem VIEW toolbar.
