# AutoTimeChn

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: AutoTimeChn

Specifies whether DIAdem uses the first channel of a channel group as the time channel or whether you specify the time channel. If the value is TRUE , DIAdem uses the first channel of a channel group as the time channel. If the input channels are waveform channels, DIAdem uses the time part of the waveform.
