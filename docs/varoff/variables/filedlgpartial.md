# FileDlgPartial

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: FileDlgPartial

Specifies whether to save only specified data or all data.

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img height="25" src="image/note.gif" width="26"/></td><td><strong>Note</strong>  The <span class="Monospace">FileDlgPartial</span> variable receives the value <span class="Monospace">TRUE</span> if you click <strong>Selective loading</strong> to close the dialog box for loading files, or if you select <strong>Only elements selected in the Data Portal</strong> in the dialog box for saving data. Otherwise, the variable is <span class="Monospace">FALSE</span>.</td></tr></table>
</div>
