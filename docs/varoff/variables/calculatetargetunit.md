# CalculateTargetUnit

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: CalculateTargetUnit

Specifies the unit symbol for the result of the calculation. If you specify the "Default" value, DIAdem uses the matching default unit of the active unit set of the result channel. If you specify an empty string, DIAdem specifies the unit symbol.
