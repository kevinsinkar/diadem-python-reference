# ApplicationChnXYRelationBinding

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ApplicationChnXYRelationBinding

Specifies whether when plotting xy-channels DIAdem automatically determines the x-channel if it is not explicitly specified. DIAdem saves this setting in the desktop file.
