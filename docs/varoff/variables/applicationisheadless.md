# ApplicationIsHeadless

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ApplicationIsHeadless

Specifies whether a DIAdem application was started without using an interface. A DIAdem application is, for example, started in TDM Server without interface, or you start a DIAdem application as Worker object without interface.
