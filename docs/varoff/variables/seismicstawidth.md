# SeismicSTAWidth

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: SeismicSTAWidth

Specifies the time span for calculating the short-term average (STA) in seconds. When calculating the STA mean value, outliers like a single impact have only a minor effect, while persistently high amplitudes of a P wave result in a high mean value.
