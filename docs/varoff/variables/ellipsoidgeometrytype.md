# EllipsoidGeometryType

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: EllipsoidGeometryType

Specifies whether DIAdem assumes that the Earth is a sphere or ellipsoid according to the 1984 World Geodetic System (WGS 84).
