# PrintRangeType

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: PrintRangeType

Specifies whether DIAdem prints all pages of a report or only certain pages of a report.

<tr>
<td class="Icon"><img height="25" src="image/note.gif" width="26"/></td><td><strong>Note</strong>  If you want to print only certain pages, use the <a href="../printfrompage/">PrintFromPage</a> variable for the first page, the <a href="../printtopage/">PrintToPage</a> variable for the last page, and assign the value <span class="Monospace">PageEnums</span> to the <a href="./">PrintRangeType</a> variable.</td>
