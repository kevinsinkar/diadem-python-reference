# ChnPairMode

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ChnPairMode

Specifies whether the numeric channel pair is real and imaginary part or magnitude and phase. In non-quantity-based calculations , DIAdem expects the phase in rad .
