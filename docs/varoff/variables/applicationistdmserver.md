# ApplicationIsTDMServer

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ApplicationIsTDMServer

Specifies whether a DIAdem application was started in TDM Server. If you test a script containing this variable in the server configuration dialog boxes, this variable also returns TRUE .
