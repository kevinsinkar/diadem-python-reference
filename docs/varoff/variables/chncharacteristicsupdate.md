# ChnCharacteristicsUpdate

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ChnCharacteristicsUpdate

Specifies whether DIAdem calculates the characteristic values when loading defined criteria automatically according to specific criteria or not. The default value is FALSE , which specifies that DIAdem does not calculate the characteristic values.
