# FileImportFilter

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: FileImportFilter

Specifies the DataPlugin which DIAdem uses for loading a data file into the Data Portal.

In the script recorded in the script editor, you then receive the value of the variable as a parameter of the <a href="../../../comoff/commands/datafileload/">DataFileLoad</a> command.</td></tr></table>
