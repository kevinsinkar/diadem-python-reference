# SINADResult

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: SINADResult

Receives the SINAD value (signal to noise and distortion). The SINAD value specifies the signal to distortion ratio in dB. The ratio is defined as the ratio of the effective energy of the input signal to the effective energy of input signals less the energy of the fundamental frequency.
