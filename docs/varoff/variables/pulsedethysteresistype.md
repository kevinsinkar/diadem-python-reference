# PulseDetHysteresisType

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: PulseDetHysteresisType

Specifies whether DIAdem interprets the hysteresis as an absolute numeric value or as a percentage. If you select Percentage , the hysteresis refers to the range between the global minimum and maximum of the current tachometer channel.
