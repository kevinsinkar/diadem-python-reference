# PartToReserve

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: PartToReserve

Specifies the percentage of messages at the end of a logfile, which DIAdem does not delete when it resets.

## Python example

```python
dd.PartToReserve  =  10
dd.LogFileReset   = 200
dd.LogfileBrowser()
```
