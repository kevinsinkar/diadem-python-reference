# ChnFindStartIdx

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ChnFindStartIdx

Specifies the row number at which DIAdem begins checking the data channel for a condition. By default DIAdem checks the data channel in the command ChnFind from the first row and in the command ChnFindReverse from the last row.
