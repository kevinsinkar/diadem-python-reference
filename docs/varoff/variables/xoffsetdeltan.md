# XOffsetDeltaN

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: XOffsetDeltaN

Receives in a cross correlation the number of values of the maximum displacement of the second xy-curve to the first xy-curve. If the XOffsetDeltaN variable receives, for example, the value 4 , you must move the second y-channel up four values in the channel table so that the y-values correspond.
