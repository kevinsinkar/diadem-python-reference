# NonLinearFitInitCoef

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: NonLinearFitInitCoef

Specifies the initially received values for the coefficients of the setup function of a non-linear curve fitting. The success of the curve fitting depends on how close the initial values are to the result.
