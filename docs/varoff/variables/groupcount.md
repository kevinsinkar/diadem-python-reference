# GroupCount

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: GroupCount

Specifies the number of channel groups in the Data Portal. You also can use the object-oriented interface to access internal data. Use ChannelGroups . Count from the script interface for internal data to realize the functionality of the GroupCount variable.
