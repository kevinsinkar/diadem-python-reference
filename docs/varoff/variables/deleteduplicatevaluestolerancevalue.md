# DeleteDuplicateValuesToleranceValue

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: DeleteDuplicateValuesToleranceValue

Specifies the tolerance value as an absolute value or as a percentage. The percentage value refers to the current channel value of the channel to be reduced. By default, DIAdem does not define a tolerance.
