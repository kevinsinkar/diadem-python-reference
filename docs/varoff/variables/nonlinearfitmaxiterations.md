# NonLinearFitMaxIterations

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: NonLinearFitMaxIterations

Specifies the maximum number of the fitting routine iterations of a non-linear curve fitting. If the number of iterations exceeds this value, the fitting process stops.
