# D3ConvertTolAbsY

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: D3ConvertTolAbsY

Specifies the absolute tolerance within which DIAdem combines y-values into a single value.

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img height="25" src="image/note.gif" width="26"/></td><td><strong>Note </strong> If the tolerance you select is too great, DIAdem combines values that should be assigned to different numbers. If the tolerance is too small, DIAdem assigns identical numbers to different representatives.</td></tr></table>
</div>
