# DeleteDuplicateValuesToleranceType

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: DeleteDuplicateValuesToleranceType

Specifies whether to define tolerance as a percentage or absolute or whether no tolerance is defined. The percentage value refers to the current channel value of the channel to be reduced. By default, DIAdem does not define a tolerance.
