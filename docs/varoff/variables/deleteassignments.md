# DeleteAssignments

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: DeleteAssignments

Specifies that DIAdem deletes the assignments for the AssignmentChnToNumericChn command and replaces the numeric channel with the assignment channel for the ChnToAssignmentChn command and also that DIAdem deletes the channel from where the assignment texts originate. The default value of the variable is FALSE .
