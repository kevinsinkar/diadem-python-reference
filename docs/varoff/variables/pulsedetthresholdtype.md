# PulseDetThresholdType

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: PulseDetThresholdType

Specifies whether DIAdem interprets the threshold value as an absolute numeric value or as a percentage. If you select Percentage , the threshold refers to the range between the global minimum and maximum of the current tachometer channel.
