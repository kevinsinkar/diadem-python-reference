# FileDlgExt

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: FileDlgExt

Receives the name extension of the file that you selected in the dialog box for loading and saving.

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img height="25" src="image/note.gif" width="26"/></td><td><strong>Note</strong>  If you call the command <a href="../../../comoff/commands/filenameget/">FileNameGet</a> with the parameter <span class="Monospace">ANY</span> and if you have selected the checkbox <strong>Hide extensions for known file types</strong> in the Windows Explorer, the <span class="Monospace">FileDlgExt</span> variable receives an empty string.</td></tr></table>
</div>
