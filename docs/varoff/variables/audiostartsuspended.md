# AudioStartSuspended

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: AudioStartSuspended

Specifies that the audio output does not start directly. Use the commands AudioOutSuspend and AudioOutResume to stop or to continue the audio output.
