# ChnMSortIP

!!! abstract "Variable &middot; `VarOff.chm`"
    Variable: ChnMSortIP

Specifies whether DIAdem overwrites the values of the input channels with the sorting result values. The default value is FALSE , which specifies that DIAdem does not overwrite the input channels.
