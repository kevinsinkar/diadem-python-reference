# IUnitCatalog.SaveAs

!!! abstract "Method &middot; `ScriptUnitCatalog.chm`"
    Method: SaveAs for UnitCatalog

Saves the current units catalog in a file with the extension .tuc .

## Signature

```python
obj.SaveAs(FileName)
```

## Python example

```python
dd.UnitCatalog.SaveAs(dd.ScriptWritePath + "MyCatalog.tuc")
```
