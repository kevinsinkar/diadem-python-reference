# IUnitCatalog.ShowChannelUnitAssistanceDlg

!!! abstract "Method &middot; `ScriptUnitCatalog.chm`"
    Method: ShowChannelUnitAssistanceDlg for UnitCatalog

Opens the Waveform X-Unit: Symbol Input Help dialog box or the Channel Unit: Symbol Entry Helper dialog box.

## Signature

```python
obj.ShowChannelUnitAssistanceDlg(ChannelList, IsWaveformX)
```

## Python example

```python
(dd.UnitCatalog.ShowChannelUnitAssistanceDlg("Grp1/Chn1 - Grp1/Chn4",False))
```
