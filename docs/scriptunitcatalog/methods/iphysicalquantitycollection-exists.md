# IPhysicalQuantityCollection.Exists

!!! abstract "Method &middot; `ScriptUnitCatalog.chm`"
    Method: Exists for Quantities

Checks whether a physical quantity already exists in the units catalog.

## Signature

```python
bExists = Object.Exists(Name)
```

## Python example

```python
dd.MsgBoxDisp(dd.UnitCatalog.Quantities.Exists("volume"))
```
