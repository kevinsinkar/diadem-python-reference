# IPhysicalUnitCollection.Remove

!!! abstract "Method &middot; `ScriptUnitCatalog.chm`"
    Method: Remove for Units

Deletes a unit with all the associated symbols, from the units catalog.

## Signature

```python
obj.Remove(NameOrIndex)
```

## Python example

```python
dd.UnitCatalog.Quantities("time").Units.Remove("second")
```
