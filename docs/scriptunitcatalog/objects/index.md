# Objects

6 objects from `ScriptUnitCatalog.chm`.

| Name | Summary |
| --- | --- |
| [`IPhysicalQuantity`](iphysicalquantity.md) | The Quantity object specifies a physical quantity from the units catalog. |
| [`IPhysicalUnit`](iphysicalunit.md) | The Unit object specifies a unit from the units catalog. |
| [`IUnitCatalog`](iunitcatalog.md) | The UnitCatalog object provides access to the DIAdem units catalog. You use the UnitCatalog object to create and to edit units sets that have physical quantities, units, and symbols. The UnitCatalog object is available as a global object in scripts and in dialog boxes. |
| [`IUnitSet`](iunitset.md) | The Set object provides a unit set from the units catalog. |
| [`IUnitSetQuantity`](iunitsetquantity.md) | The UsedQuantity object specifies a physical quantity that is used in a unit set of the units catalog. |
| [`IUnitSymbol`](iunitsymbol.md) | The Symbol object specifies the symbol of a unit from a unit set in the units catalog. |
