# IUnitCatalog.ActiveSet

!!! abstract "Property &middot; `ScriptUnitCatalog.chm`"
    Property: ActiveSet for UnitCatalog

Returns the enabled unit set of the units catalog. DIAdem uses the active units set as default setting in all dialog boxes if ActiveSetEnable is enabled.

## Signature

```python
return_value = obj.ActiveSet
```

## Python example

```python
dd.MsgBoxDisp (dd.Unitcatalog.ActiveSet.Name)
```
