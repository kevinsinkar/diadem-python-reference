# IPhysicalUnit.Offset

!!! abstract "Property &middot; `ScriptUnitCatalog.chm`"
    Property: Offset for Unit

Returns the offset for conversion to the base unit of a physical quantity, in the units catalog.

## Signature

```python
obj.Offset
```

## Python example

```python
print(dd.UnitCatalog.Quantities("temperature").Units("degree_celsius").Offset)
```
