# Collections

7 collections from `ScriptUnitCatalog.chm`.

| Name | Summary |
| --- | --- |
| [`AllSymbols`](allsymbols.md) | The AllSymbols collection provides all the symbols from the units catalog. |
| [`Quantities`](quantities.md) | The Quantities collection provides all the physical quantities from the units catalog. |
| [`Sets`](sets.md) | The Sets collection provides all the unit sets of the units catalog. |
| [`SymbolAliases`](symbolaliases.md) | The SymbolAliases collection provides all the symbols of a unit in the units catalog. |
| [`Units`](units.md) | The Units collection provides all the units of a physical quantity from the units catalog. |
| [`UsedQuantities`](usedquantities.md) | The UsedQuantities collection provides all the physical quantities that are used in a unit set of the units catalog. |
| [`UsedUnits`](usedunits.md) | The UsedUnits collection provides all the units of a physical quantity that are used, from a specific unit set in the units catalog. |
