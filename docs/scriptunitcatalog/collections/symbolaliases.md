# SymbolAliases

!!! abstract "Collection &middot; `ScriptUnitCatalog.chm`"
    Collection: SymbolAliases

The SymbolAliases collection provides all the symbols of a unit in the units catalog.

## Python example

```python
dd.MsgBoxDisp (dd.UnitCatalog.Quantities("length").Units("meter").SymbolAliases.Count)
```
