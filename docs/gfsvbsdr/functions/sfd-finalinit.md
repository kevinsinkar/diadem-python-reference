# SFD_FinalInit

!!! abstract "Function &middot; `Gfsvbsdr.chm`"
    Function: SFD_FinalInit

Executes a final initialization of the connected device and the variables of the script in the script driver . DIAdem calls the SFD_FinalInit function once when it completes a measurement preparation.

## Signature

```python
dd.SFD_FinalInit(ErrorP, [ScanIntervallV])
```
