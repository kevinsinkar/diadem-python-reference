# SFD_DeInitOutChannel

!!! abstract "Function &middot; `Gfsvbsdr.chm`"
    Function: SFD_DeInitOutChannel

Deinitializes an active output channel in the script driver . DIAdem calls the SFD_DeInitOutChannel function during the measurement wrap-up, once for every active output.

## Signature

```python
dd.SFD_DeInitOutChannel(ChannelNumberV, ErrorP)
```
