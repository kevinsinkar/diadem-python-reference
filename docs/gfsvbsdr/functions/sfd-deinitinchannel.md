# SFD_DeInitInChannel

!!! abstract "Function &middot; `Gfsvbsdr.chm`"
    Function: SFD_DeInitInChannel

Deinitializes the inputs used in the script driver . DIAdem calls the SFD_DeInitInChannel function during the measurement wrap-up, once for every active measurement channel.

## Signature

```python
dd.SFD_DeInitInChannel(ChannelNumberV, ErrorP)
```
