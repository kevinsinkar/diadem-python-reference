# SFD_DelInit

!!! abstract "Function &middot; `Gfsvbsdr.chm`"
    Function: SFD_DelInit

Deinitializes the connected devices and disconnects the devices in the script driver . DIAdem calls the SFD_DelInit function once during measurement follow-up tasks.

## Signature

```python
dd.SFD_DeInit(ErrorP)
```
