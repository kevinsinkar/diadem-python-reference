# ChnOrdATCalc

!!! abstract "Command &middot; `ComOff.chm`"
    Command: ChnOrdATCalc

Runs an order and frequency analysis in the time domain of a signal.

## Signature

```python
dd.ChnOrdATCalc( XW , Y , X1 )
```

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img height="25" src="image/note.gif" width="26"/></td><td><strong>Note  </strong>Use the <a href="../ordatparaset/">OrdATParaSet</a> command to set the parameters for order analysis in the time domain according to the input channels.</td></tr></table>
</div>

## Parameters

<div markdown="1">
<table class="Borderless">
<tr><td width="150"><em>XW</em></td>
<td>Specifies the data channel containing the time values of the signal to be inspected.</td></tr>
<tr><td width="150"><em>Y</em></td>
<td>Specifies the data channel containing the amplitude values of the signal to be inspected.</td></tr>
<tr><td width="150"><em>X1</em></td>
<td>Specifies the data channel that contains the rpm values of the signal you want to analyze.</td></tr>
</table>
</div>
