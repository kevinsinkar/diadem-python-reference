# DataChangedClear

!!! abstract "Command &middot; `ComOff.chm`"
    Command: DataChangedClear

Marks the internal data area as unchanged. The DataChanged variable receives information about whether the internal data area contains modified data.

## Signature

```python
dd.DataChangedClear()
```
