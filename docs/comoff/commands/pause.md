# Pause

!!! abstract "Command &middot; `ComOff.chm`"
    Command: Pause

Stops a script for a specific period of time in seconds. DIAdem displays the remaining pause time on the status bar.

## Signature

```python
dd.Pause( PauseDuration )
```
