# MsgBoxStateResetAll

!!! abstract "Command &middot; `ComOff.chm`"
    Command: MsgBoxStateResetAll

Resets the setting for all dialog boxes so that some dialogs are only displayed when the Shift key is pressed. After executing this command, DIAdem displays the dialog boxes again for which you selected the option Only display this message when the Shift key is pressed .

## Signature

```python
dd.MsgBoxStateResetAll()
```
