# ChnRelativize

!!! abstract "Command &middot; `ComOff.chm`"
    Command: ChnRelativize

Determines for each channel value the percentage of the total sum of all channel values.

## Signature

```python
return_value = dd.ChnRelativize( Y , ResultChannel )
```

## Parameters

<div markdown="1">
<table class="Borderless">
<tr><td width="150"><em>Y</em></td>
<td>Specifies the data channel.</td></tr>
<tr><td width="150"><em>ResultChannel</em></td>
<td>Specifies the result channel.</td></tr>
</table>
</div>
