# PrintMaxScale

!!! abstract "Command &middot; `ComOff.chm`"
    Command: PrintMaxScale

Specifies the minimum margins for the current printer and the specified page format. Call this command before calling other print commands.

## Signature

```python
dd.PrintMaxScale([WndName])
```
