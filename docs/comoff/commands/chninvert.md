# ChnInvert

!!! abstract "Command &middot; `ComOff.chm`"
    Command: ChnInvert

Mirrors the values in a data channel. DIAdem exchanges the first channel value with the last value and so on.

## Signature

```python
dd.ChnInvert( X )
```

## Parameters

<div markdown="1">
<table class="Borderless">
<tr><td width="150"><em>X</em></td>
