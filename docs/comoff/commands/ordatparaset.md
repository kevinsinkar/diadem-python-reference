# OrdATParaSet

!!! abstract "Command &middot; `ComOff.chm`"
    Command: OrdATParaSet

Sets the parameters for Order analysis in the time domain according to the input channels.

## Signature

```python
dd.OrdATParaSet( XW , Y , X1 )
```

## Parameters

<div markdown="1">
<table class="Borderless">
<tr><td width="150"><em>XW</em></td>
<td>Specifies the data channel that contains the time values of the signal you want to analyze.</td></tr>
<tr><td width="150"><em>Y</em></td>
<td>Specifies the data channel containing the amplitude values of the signal you want to analyze.</td></tr>
<tr><td width="150"><em>X1</em></td>
<td>Specifies the data channel containing the rpm values of the signal you want to analyze.</td></tr>
</table>
</div>
