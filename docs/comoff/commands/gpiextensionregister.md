# GPIExtensionRegister

!!! abstract "Command &middot; `ComOff.chm`"
    Command: GPIExtensionRegister

Registers a GPI extension in DIAdem. It only registers for the running DIAdem instance and is not saved in the desktop file. The registered DLL is not displayed in the GPI Extensions: User Extensions dialog box.

## Signature

```python
dd.GPIExtensionRegister(FileName)
```
