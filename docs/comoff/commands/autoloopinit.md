# AutoLoopInit

!!! abstract "Command &middot; `ComOff.chm`"
    Command: AutoLoopInit

Starts a progress display in the status bar.

## Signature

```python
dd.AutoLoopInit()
```

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img height="25" src="image/note.gif" width="26"/></td><td><strong>Note</strong>  During the progress display, DIAdem suppresses the progress display of the DIAdem commands.</td></tr></table>
</div>

## Parameters

<div markdown="1">
<table class="Borderless">
<tr><td>None</td></tr>
</table>
</div>

## Python example

```python
dd.AutoLoopInit()
dd.Pause(10)
```
