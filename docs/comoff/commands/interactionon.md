# InteractionOn

!!! abstract "Command &middot; `ComOff.chm`"
    Command: InteractionOn

Enables the interaction mode.

## Signature

```python
dd.InteractionOn([InteractionText])
```

## Notes

<div markdown="1">
<table class="Borderless"><tr><td class="Icon"><img height="25" src="image/note.gif" width="26"/></td><td><strong>Note</strong>  You cannot record scripts in the interaction mode.</td></tr></table>
<table class="Borderless">
<tr><td class="Icon"><img height="25" src="image/note.gif" width="26"/></td><td><strong>Note</strong>  You must not use the <span class="Monospace">InterActionOn</span> command when a modal dialog box is open.</td></tr></table>
</div>
