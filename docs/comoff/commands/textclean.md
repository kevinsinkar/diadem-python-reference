# TextClean

!!! abstract "Command &middot; `ComOff.chm`"
    Command: TextClean

Removes all characters that cannot be printed, from a string. In the 7-bit ASCII code the characters that cannot be printed are the values 0 to 31 and additionally in the unicode character-set the values 127, 129, 141, 143, 144, and 157.
