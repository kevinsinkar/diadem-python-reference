# DIAdem Python Scripting Reference (MkDocs)

A consolidated, browseable Python reference for the DIAdem COM Automation API,
generated from the DIAdem 2026 Q2 (Edition April 2026) help bundle.

## Layout

```
diadem-mkdocs/
├── mkdocs.yml          # MkDocs Material config (auto-generated)
├── docs/               # Source: 13,558 markdown files
│   ├── index.md
│   ├── getting-started.md
│   └── <chm-slug>/
│       ├── index.md
│       └── <kind>/
│           ├── index.md
│           └── <topic>.md
└── site/               # Generated by `mkdocs build`
```

## Local browsing

```bash
# from c:/Users/ksinkar/Documents/DIAdem Files/Help/diadem-mkdocs
../.venv/Scripts/python.exe -m mkdocs serve
```

Open http://127.0.0.1:8000/ in a browser. The dev server gives you live reload,
instant navigation, dark mode toggle, and full-text search across all topics.

To produce a static folder (already done once):

```bash
../.venv/Scripts/python.exe -m mkdocs build
```

The output goes to `site/`. You can open `site/index.html` directly in a
browser, or copy `site/` to a network share / IIS / GitHub Pages / S3.

## Regenerating from updated CHMs

If DIAdem ships a new help bundle:

```bash
# from c:/Users/ksinkar/Documents/DIAdem Files/Help
.venv/Scripts/python.exe convert_chm_to_html.py        # decompile CHMs
.venv/Scripts/python.exe build_python_reference.py --tier 12
```

This regenerates `diadem-python-reference.html`, `diadem.pyi`, `diadem-mkdocs/`,
and `diadem-python-coverage.json` together. Skip individual outputs with
`--skip-html`, `--skip-pyi`, or `--skip-mkdocs`.
